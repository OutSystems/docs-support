OutSystems Developer Cloud 2025-11-17

Released on Nov 17, 2025

New

  • Workflow Editor: Keyboard Shortcuts Workflow
    Editor now supports keyboard shortcuts, significantly improving development speed and productivity on both Windows and macOS operating systems.
    A complete list of all available shortcuts can be quickly accessed by clicking the question mark icon (?) in the top bar and selecting the new option. (RDODCT-3621)

Bug Fixing

  • Resolved an issue where, in the settings detail page of ODC Portal, downloading a Binary setting was incorrectly failing or resulting in an error. (RDLOT-4064)
  • Fixed an issue in the Link widget that included the wrong style classes. (RPM-6301)
