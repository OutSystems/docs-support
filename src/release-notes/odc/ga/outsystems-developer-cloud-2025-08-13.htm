OutSystems Developer Cloud 2025-08-13

Released on Aug 13, 2025

New

  • Access to Add-On-related features in the ODC Portal is now controlled by the subscription status of their respective Add-Ons:
    • IP Filters is now available only with an active Sentry subscription.
    • Private Gateways is now available only with an active Private Gateways subscription.
    (RDLOT-3873)
  • A six-digit format will now be used for self-registration codes. (RDPIB-2789)
