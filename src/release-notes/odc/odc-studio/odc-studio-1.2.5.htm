ODC Studio 1.2.5

Released on Oct 10, 2023

Bug Fixing

  • Fixed an issue that caused the schedule timer view to have a misplaced label. (RDEV-5967)
  • Fixed an issue that caused some Debugger Pane buttons to be unaligned for applications. (RDEV-5969)
  • Fixed a crash on IDE startup when clicking to connect while automatic login is in progress. (RDOIBT-1836)
  • Fixed an issue that prevented automatically change of theme colour on changing primary colour of the application. Now, theme colour will change automatically on changing primary colour of the application. (RDPMBT-3626)
