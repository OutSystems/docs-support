ODC Studio 1.5.5

Released on Jan 20, 2025

New in ODC Studio 1.5.5

  • Improved the indicator of unsaved changes after performing undo/redo operations. (RDEV-7779)
  • You can now scroll on the flow editor by holding the middle mouse button and moving the mouse. (RDEV-8071)
  • It is now possible to import multiple images and resources in a single operation. (RDEV-8220)
