MABS 2024-12-11

Released on Dec 11, 2024

New

  • The Android permissions associated with the upload widget are no longer added by default (RDMR-299)
  • Updates Android tools to the latest stable versions. (RDMR-412)
  • Upgrade Xcode to version 16.1. (RNMT-6894)

Bug Fixing

  • Fixes an issue where some applications with some special characters in the name would fail to build on iOS. (RNMT-6887)
