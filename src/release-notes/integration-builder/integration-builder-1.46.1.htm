Integration Builder 1.46.1

This is a preliminary version of the document.

New in Integration Builder 1.46.1

  • Improved the message associated with error code OS-INBL-API-00004. (RCNT-6613)

Bug Fixing

  • Fixed an issue that caused external database entity attributes to become sorted alphabetically when an integration was republished. (RCNT-6618)
  • Fixed an issue that caused columns with custom data types, in relational database integrations, to revert to their original types when the integration was edited. (RCNT-6635)
