OutSystems Developer Cloud 2026-03-17

Released on Mar 17, 2026

New

  • When editing an existing AI model endpoint, changing any configuration field (other than the name) now requires all credential fields to be provided before you can test or save. Previously, re-entering credentials was only required when changing the endpoint URL. This does not affect creating new endpoints or renaming existing ones. (RDCCP-2444)
