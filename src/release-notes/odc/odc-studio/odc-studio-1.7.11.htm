ODC Studio 1.7.11

Released on Jun 15, 2026

New in ODC Studio 1.7.11

  • ODC Studio has been redesigned with a modernized, distraction-free workspace — featuring a modular panel layout, refreshed typography, and a new iconography system optimised for HiDPI and Retina displays.

    (RDEV-9659)

  • The Mentor input field now allows to write until 12 lines without needing to scroll.

    (RDEV-9704)

Bug Fixing

  • Fixed an issue where content copied in the editor could not be pasted back into the application.

    (RDEV-9657)

  • Fixed an issue that would cause an error when copying and pasting a large amount of objects.

    (RDEV-9691)

  • Fixed an issue where clicking Stop during a Mentor conversation could make the IDE unresponsive.

    (RDEV-9731)

  • Fixed an issue where keyboard shortcuts would stop working on trees.

    (RDEV-9755)

  • Fixed an error that occurred when opening a consumer and cancelling the operation before it finished.

    (RDEV-9759)

  • Fixed an issue where hidden attributes in an aggregate preview could not be shown.

    (RDEV-9839)
  • Fixed a crash that could occur during merge operations when global event handlers had unresolved matches in the merge tree. (RPM-6655) Service Studio Merge operation
  • Fixed an issue where Compare and Merge could cause an unexpected error, closing Studio. (RPM-6868) Service Studio Merge operation
