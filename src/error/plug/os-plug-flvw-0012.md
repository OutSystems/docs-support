# OS-PLUG-FLVW-0012

## Error message

"The download failed."

## Platform

iOS

## Cause

The plugin failed to download the file from the specified URL. Possible causes include network connectivity issues, an invalid or unreachable URL, a server error, or a request timeout.

## Impact

The document cannot be opened from the given URL.

## Recommended action

Verify the URL is accessible and the server is responding correctly. If the error is intermittent, retry the operation.

If the problem persists, open a support case with [OutSystems support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-PLUG-FLVW-0012).
