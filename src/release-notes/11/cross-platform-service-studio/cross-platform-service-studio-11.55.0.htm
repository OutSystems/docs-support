Cross-platform Service Studio 11.55.0

Released on Dec 16, 2024

New in Cross-platform Service Studio 11.55.0

  • It is now possible to right-click on an Entity or Entity Action to instantly create wrappers with basic validations. (RDEV-8087)
  • It is now possible to filter by author and version in "Open other version". (RDEV-8120)

Bug Fixing

  • Fixed an issue that caused duplicate pasting while using screen and flow editors. (RDEV-8085)
  • Fixed an issue that caused a warning for unused resource to be shown when the resource was only being used in extensibility configurations. (RDEV-8114)
  • Fixed an issue where double-clicking items in the tree could cause the application to become unresponsive. (RDEV-8143)
