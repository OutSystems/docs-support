Platform Server 11.41.2

New in Platform Server 11.41.2

  • Updated Swagger UI dependency lodash to version 4.17.23. This update has no runtime impact.

    (R11DT-3616)

  • Updated Swagger UI dependencies dompurify (3.4.0) , immutable (3.8.3) , lodash (4.18.1) and swagger-client (3.37.2) to ensure continued security compliance. This update has no runtime impact.

    (R11DT-3830)

Bug Fixing

  • Fixed a vulnerability in the App Feedback system component that could allow unauthorized outbound requests. Improved sanitization of user-submitted content. Feedback elements using URIs (e.g. images, external styles, iframe sources, ...) will not be retrieved by the server, which can impact their preview. If their content is filled at the time of submission and does not rely on external sources, there is no change in preview behavior. (RPM-6678) Application Runtime System Components
