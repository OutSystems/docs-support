OutSystems Developer Cloud 2025-01-17

Released on Jan 17, 2025

New

  • Improved the error message displayed when a deployment fails due to a reduction in the length of database attributes that contain incompatible data. (RPLAT-2390)

Bug Fixing

  • Fixes a bug where screens with custom URLs and boolean input parameters were not correctly passing the parameter values to the paths. This ensures proper navigation to the screens. (RAR-3147)
  • Fixed an issue where publication would fail when creating a new entity with a default value of a column set to "NullDate()". (RPLAT-2899)
