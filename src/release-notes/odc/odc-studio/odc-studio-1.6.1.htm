ODC Studio 1.6.1

Released on Oct 07, 2025

New in ODC Studio 1.6.1

  • Built-in structures and built-in static entities are now visible in the tree, allowing users to easily inspect these core components. (RDEV-8744)

Bug Fixing

  • Fixed issue that prevents Structured output nodes with differences from being automatically expanded in merge. (RDEV-8765)
  • Fixed an issue where action calling changes were not correctly highlighted in the Merge Editor. (RDEV-8783)
  • Fixed an unexpected error that happens when undoing a change in the Call Agent after it has been popped out or docked. (RDEV-8786)
  • Fixed a crash that would lead to the ODC Studio getting closed without warning. (RDEV-8797)
  • Fixed an unexpected error that occurred when a structured attribute was clicked in merge editor. (RDEV-8813)
  • Fixed an issue that would cause an error when opening a new tab. (RDEV-8818)
