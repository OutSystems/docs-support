OutSystems Developer Cloud 2026-07-02

Released on Jul 02, 2026

New

  • Workflow Editor: Scheduled Triggers

    • You can now configure workflows to start automatically on a daily, weekly, or monthly schedule in UTC without requiring an external event.

    • Key Features:

      • Daily, weekly, or monthly scheduling with flexible time selection (5-minute intervals)

      • Canvas summary displays your configured schedule (e.g., "Run daily at 09:00, 14:00 UTC").

      • Schedule-triggered instances are identifiable in workflow logs.

      • How to configure:

        • Select the Schedule tab in the Start node of your workflow, choose your frequency, configure times, and publish. All schedules run in UTC.

      (RDODCT-4409)

    Bug Fixing

    • Fixed an issue where end-user roles remained in the UI and APIs after an app was undeployed from an environment. Now, when an app is undeployed from a stage, ODC automatically removes its end-user roles from that environment. Roles in other stages where the app remains deployed are unaffected. (RPM-6799)
    • Network requests failed due to poor network conditions no longer cause the user's session to be lost. (RPM-6967)
