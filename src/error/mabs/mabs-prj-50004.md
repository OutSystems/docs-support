# OS-MABS-PRJ-50004

## Error message

`The code project structure is invalid. If the problem persists, open a support case.`

## Cause

This is an internal error. A file or folder in the native project template is missing or has an unexpected structure.

## Impact

The app package can't be generated.

## Recommended action

Regenerate the app package. This might be a temporary issue that resolves on its own.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MABS-PRJ-50004).
