ODC Studio 1.5.1

This is a preliminary version of the document.

New in ODC Studio 1.5.1

  • About, Preferences and Shortcut dialogs should now be displayed in the same screen where the main window is displayed. (RDEV-8036)
  • Removed the selection limit in 'Add public elements', allowing users to select over 100 elements at once. (RDEV-8074)
  • Improved 'Add Public Elements' to exclude elements from the same app. (RDEV-8122)
  • Styling improvements to post login and logout pages. (RDEV-8123)

Bug Fixing

  • Fixed an issue that caused connectivity issues in some network configurations. (RDEV-8168)
