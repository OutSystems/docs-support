LifeTime Management Console 11.17.2

Released on Jul 05, 2023

New in LifeTime Management Console 11.17.2

  • OutSystems Charts application was removed from LifeTime solution. This will avoid conflicts with recent versions of OutSystems Charts that may be deployed by the customer on environments running LifeTime. (R11CT-2204)
  • Improved the user experience in the list of Application Versions on the Deployment Plan page by making it clearer which was the previous application version and which application version was updated to the target environment. (R11CT-2206)
  • Platform Server version updated to 11.21.0 (R11CT-2215)
  • Now, the password change functionality has been moved to a convenient pop-up, initiated by clicking the "Change Password" button, ensuring a seamless user experience and eliminating any confusion regarding password updates on the User Edit screen. (RDOI-279)

Bug Fixing

  • Fixed an issue that caused an error when staging a producer of an ISV-protected module in LifeTime. (RPM-3820) Application Lifecycle LifeTime
  • Fixed an issue that caused modules to be deleted in the target environment of a deployment plan. This occurred when there was an issue that would stop the compiler service during the synchronization of application versions in LifeTime, marking the affected versions as non existing in the source environment. After performing a deployment to the target environment, these modules were deleted. (RPM-3956) Application Lifecycle LifeTime
  • Fixed an issue in the LifeTime API that caused exceptions like "Invalid status change : trying to change Staging with the id..." being logged when calling the method "GET /deployments/{DeploymentKey}/". This occurred when the deployment was in certain states (e.g. "Finished With Warnings") and the API method would attempt to change the deployment state. (RPM-3993) Application Lifecycle LifeTime
  • Fixed Japanese translations on ApplySettings_Popup, when applying settings to an application. (RPM-4076) Application Lifecycle LifeTime

Note

  • This version includes Platform Server 11.21.0, which is compatible with Windows Server 2022. The installation checklist you'll find in the installer package is missing Windows Server 2022 in the Operating System dropdown options. You may still install LifeTime 11.17.2 on Windows Server 2022. Please make sure you download the checklist from our website to have Windows Server 2022 in the Operating System dropdown options.
  • In this LifeTime version, manual installation of Microsoft Visual C++ 2015-2022 is required. The checklist bundled with the installer doesn't include the necessary steps for that. We acknowledge that this is a known issue, please download the checklist from our website that includes the steps for the manual installation of Microsoft Visual C++ 2015-2022.
