ODC Studio 1.6.4

This is a preliminary version of the document.

New in ODC Studio 1.6.4

  • It is now possible to select multiple libraries to update. (RDEV-8680)
  • You can now use object mapping in the input of an action in Agents 'Action calling'. (RDEV-8827)

Bug Fixing

  • Resolved a critical issue where cancelling an Operating System shutdown could cause the application to close unexpectedly. (RDEV-8834)
