OutSystems Developer Cloud 2025-08-22

Released on Aug 22, 2025

Bug Fixing

  • Accessibility improvements on Feedback Messages:
    • Feedback Messages now have their ARIA role set to "alert" by default.
    • Feedback Messages can now be highlighted with the tab key and closed with the space or enter keys.
    • Feedback Messages now have the "has-accessible-features" class set by default when the class is present somewhere in the current screen's DOM.
    (RPM-6067)
