OutSystems Developer Cloud 2026-02-18

Released on Feb 18, 2026

New

  • Workflow Editor: Copy Activity link - You can now share a direct link of an Workflow activity (node). - To use this feature, on the top right side of the activity card, on the operations menu, select option "Copy Link", which is copied to the clipboard and you can paste to where/who you need. (RDODCT-3889)

Bug Fixing

  • Previously, users invited through the ODC Portal who logged in using an External IdP without finishing onboarding had an incorrect Invited or Pending Registration status. Now, these users have an Active status, ensuring accurate user management and visibility. (RPM-6265)
