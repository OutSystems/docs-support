ODC Studio 1.7.15

Released on Jul 13, 2026

Bug Fixing

  • Fixed an issue that caused a misalignment in advanced SQL Output table. (RDEV-9893)
  • Fixed an issue in the CSS editor where the styles editor pane had a minimum size larger than the available panel space. (RDEV-9909)
  • Improved experience for unsual line terminators in expression editors. (RDEV-9912)
  • Fixed an issue that caused an unexpected error when closing a tab immediately after publishing. (RDEV-9922)
  • Fixed an issue that caused missing images when using autocomplete on monaco editor. (RDPIM-4290)
