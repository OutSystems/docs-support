ODC Studio 1.6.19

Released on Feb 09, 2026

Bug Fixing

  • Fixed an issue in merge of JavaScript nodes that caused an unintended merge when choosing 'Other Revision' instead of replacing it with the selected version. (RDEV-9102)
  • Fixed an issue that could cause an error while fetching for Screen Templates when attempting to create a new Screen. (RDEV-9127)
  • Fixed an issue where a progress dialog would appear prematurely during a merge, preventing users from selecting files in the file dialog. (RDEV-9165)
