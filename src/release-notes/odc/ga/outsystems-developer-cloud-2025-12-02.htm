OutSystems Developer Cloud 2025-12-02

Released on Dec 02, 2025

New

  • Users with the <b>Manage API Clients</b> permission no longer automatically receive the <b>Manage organization roles</b> permission. This update strictly enforces the principle of least privilege, ensuring users do not gain unintended access to organization role management functions. (RDPIB-3017)
