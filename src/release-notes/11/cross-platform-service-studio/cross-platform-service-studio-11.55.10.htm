Cross-platform Service Studio 11.55.10

Released on Feb 24, 2025

New in Cross-platform Service Studio 11.55.10

  • Improved the experience of closing Studio by showing a single confirmation dialog listing all apps with unsaved changes. (RDEV-8248)

Bug Fixing

  • Fixed an issue that caused Text Editor to be available for Name and Label properties of 'Automatic Activity' and 'Decision' process activities. (RDEV-8294)
  • Fixed an issue that prevented a debug session from starting if the module was moved to a different app by another user and the module was opened from the Recent modules section. (RDEV-8300)
  • Fixed an issue that caused an unexpected error when concurrent users debug the same application. (RDEV-8311)
  • Fixed an issue that could cause an unexpected error when adding a secondary module to an app and the home module of the app had its stylesheet reset through Theme Editor. (RPM-3895) Service Studio Windows and Editors
  • Fixed an issue that would cause the login window to appear when it was not needed. (RPM-4977) Service Studio Authentication
