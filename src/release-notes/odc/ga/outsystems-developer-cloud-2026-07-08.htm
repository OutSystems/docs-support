OutSystems Developer Cloud 2026-07-08

Released on Jul 08, 2026

New

  • You can now modify stage display names and reorder non-production stages in the Organization area of the ODC Portal. This update also introduces new Manage Portfolio and Manage Stage permissions. (RDLOT-4664)
  • Added Request_GetMethod and Request_GetHeaders to the HTTP System Extension, giving ODC developers access to the request's HTTP verb and headers in their logic. (RTAFB-9185)
