OutSystems Developer Cloud 2026-06-02

Released on Jun 02, 2026

Bug Fixing

  • Fixed an issue that affected the accurate display of Timer revisions in the Logs list. Previously, discrepancies could occur when an app revision did not align with its corresponding timer revision. This enhancement, identified as RPM-6385, ensures that the app and timer revisions now match consistently, providing a clearer and more reliable logging experience. (RPM-6385)
  • Fixed an issue that previously hindered the ability to filter logs by agents. With this enhancement, identified as RPM-6603, users can now seamlessly access agent-specific data from both the assets filter and the log list. This update aims to provide more efficient navigation and improved visibility of agent-related information in the Logs screen. (RPM-6603)
