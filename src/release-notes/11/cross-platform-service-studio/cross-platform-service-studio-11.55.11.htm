Cross-platform Service Studio 11.55.11

Released on Mar 03, 2025

Bug Fixing

  • Fixed an issue affecting Windows users where OutSystems application files (oap) were not set to open with Service Studio. (RDEV-8314)
  • Fixed an issue that prevented the placement of the cursor when editing a comment node content. (RDEV-8321)
  • Fixed an issue where the Studio failed to highlight the property when navigating to an error. (RDEV-8328)
  • Fixed an error message in aggregates to enhance clarity by removing the 'above' keyword. (RDEV-8330)
  • Fixed an issue that caused an error starting a debug session. (RDEV-8333)
  • Fixed an issue that could cause an unexpected error after opening a module. (RDEV-8334)
  • Fixed an issue that caused an unexpected error when adding 'show-default-value' attribute to Input widgets. (RPM-2741) Service Studio
