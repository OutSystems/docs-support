OutSystems Developer Cloud 2025-07-07

Released on Jul 07, 2025

New

  • Now, the role of the user opening a workflow activity will be displayed, if the workflow is assigned to that role (RDASS-2345)
  • We've launched the App Analytics Stream add-on. This new feature enables the ODC application telemetry - logs, traces, and metrics - to be streamed in near real-time to your preferred Application Performance Monitoring (APM) tools using the OpenTelemetry format. It allows you to explore advanced monitoring use cases, enhancing your application observability. (RDUOPT-4594)
