ODC Studio 1.5.2

This is a preliminary version of the document.

New in ODC Studio 1.5.2

  • Improved the login interface to include an organization example and information on where to find the organization subdomain. (RDEV-8029)
  • About, Preferences and Shortcut dialogs should now be displayed in the same screen where the main window is displayed. (RDEV-8036)
  • Elements of the same kind can now be enclosed within a folder in the tree. (RDEV-8039)

Bug Fixing

  • Fixed an issue that caused an unexpected error when the session expires. (RDEV-8158)
  • Fixed an issue that occasionally caused the application to become temporarily unresponsive during auto-save operations. (RDEV-8173)
  • Fixed an issue that caused the application to become unresponsive when closing. (RDEV-8174)
  • Fixed an issue that caused an unexpected error in ODC Studio when an error occurred while fetching app/library templates. When this happens, the dialog should now display an error message. (RDEV-8175)
  • Fixed an issue that caused an unexpected error when generating entity action wrappers. (RDEV-8176)
