Conversion Assessment Tool 2.0.1.3

Released on Dec 16, 2025

New in Conversion Assessment Tool 2.0.1.3

  • Introduction of a new Data Warning for large text blob on entities attributes (over 100Mb) (RADRT-4004)
  • Added a plan dependencies graph view mode to better understand how all conversion plans are related to each other (RADRT-4159)
  • SSO login flow now supports the signum claim as an alternative to preferred_username when identifying users during authentication. (RADRT-4235)

Bug Fixing

  • Modules with deleted multi-tenant elements are no longer flagged for using multi-tenancy. (RADRT-4149)
  • Resolved a timeout issue that occurred when accessing the auto-map feature for the first time. (RADRT-4150)
