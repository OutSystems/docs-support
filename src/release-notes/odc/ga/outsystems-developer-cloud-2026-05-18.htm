OutSystems Developer Cloud 2026-05-18

Released on May 18, 2026

New

  • The Logs console now allows you to filter the logs by agent. Pick an agent from the Assets dropdown and the console returns only logs from that agent. Apps and agents sit side-by-side in the same control, so you can switch workload types without leaving the page. The Traces console keeps the app-only filter, since agents always run as spans inside an app's trace.

    (RDASS-3835)

  • Mentor AppGen:

    • You can now browse and select external entities and themes directly within the conversational interface, streamlining the app configuration experience.
    • Mentor now generates login screens with Identity Provider (IdP) support, enabling enterprise-grade authentication flows in your generated applications.
    • Users can now stop an ongoing AI conversation at any time using the new Stop button in the Chat Pane, giving you more control over the generation process.
    • Mentor now generates View and Edit screens assembled more accurately based on your data model, reducing the need for manual corrections.
    • You can now select the application theme directly in the Blueprint phase before generating your app.
    • Mentor now validates theme compatibility at the start of app generation, preventing errors and ensuring better quality results.
    • Tailor your app's look from the start - When building with the OutSystems UI framework, you can now instruct Mentor to generate your app with a dark theme
    (RDMOR-3850)
