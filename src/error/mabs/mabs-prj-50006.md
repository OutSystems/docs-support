# OS-MABS-PRJ-50006

## Error message

`An error occurred while preparing the project template. If the problem persists, open a support case.`

## Cause

This is an internal error. The iOS or Android project template archive is missing or couldn't be prepared.

## Impact

The app package can't be generated.

## Recommended action

Regenerate the app package. This might be a temporary issue that resolves on its own.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MABS-PRJ-50006).
