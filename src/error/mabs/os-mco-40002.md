# OS-MCO-40002

## Error message

`Extensibility Configurations are missing image <Identifier> in <Where>.`

## Cause

Your mobile app's extensibility configurations are missing an image from a custom icon or splash screen image catalog.

For example, using Universal Extensibility:

```json
{
  "buildConfigurations": {
    "splashscreens": {
      "ios": {
        "imageCatalog": [
          { "identifier": "Default@2x~universal~anyany", "source": "$images.IosSplash2xUniversalAnyany" },
          { "identifier": "Default@2x~universal~comany", "source": "$images.IosSplash2xUniversalComany" },
          { "identifier": "Default@2x~universal~comcom", "source": "$images.IosSplash2xUniversalComcom" }
          // Missing the other images
        ]
      }
    }
  }
}
```

Another example, using Cordova-based Extensibility:

```json
{
  "icons": {
    "android": [
      { "density": "ldpi", "resource": "$images.AndroidIconLdpi" },
      { "density": "mdpi", "resource": "$images.AndroidIconMdpi" },
      { "density": "hdpi", "resource": "$images.AndroidIconHdpi" }
      // Missing xhdpi, xxhdpi, xxxhdpi
    ]
  }
}
```

## Impact

The mobile compilation fails, causing the native mobile build to fail.

## Recommended action

Add the complete set of images for the icon or splash screen image catalog to your mobile app's extensibility configurations. Alternatively, remove the custom icons and splash screens.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MCO-40002).

## More info

For detailed information about the required sizes and densities for icons and splash screens, refer to [Icon sizes and densities](https://success.outsystems.com/documentation/11/deploying_apps/mobile_apps_packaging_and_delivery/customize_your_mobile_app/modify_the_app_icon/#icon-sizes-and-densities) and [Device screen sizes and densities](https://success.outsystems.com/documentation/11/deploying_apps/mobile_apps_packaging_and_delivery/customize_your_mobile_app/use_custom_splash_screens/#devices-screen-sizes-and-densities). Use this documentation only for image sizes and densities, not for how to provide or reference images.
