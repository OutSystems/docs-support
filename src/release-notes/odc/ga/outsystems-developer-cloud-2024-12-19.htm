OutSystems Developer Cloud 2024-12-19

Released on Dec 19, 2024

New

  • You can now disable all Traces (server and client-side) and Analytics for specific apps, reducing noise in your monitoring view. Logs remain available, and you can re-enable these features anytime for deeper troubleshooting, giving you control and flexibility to focus on what matters most. (RDLOT-3357)
  • Workflow Editor: Icon View
    • The Workflow Editor we have now includes an Icon view.
    • There are now two options to visualize a workflow: Summary view (existent) or a Icon view (new).
    (RDODCT-2845)

Bug Fixing

  • Improved Service Worker caching mechanism, ensuring Service Worker updates are better reflected in the 1CP. (RAR-3054)
  • Fixed an issue in external authentication where sensitive data could sometimes stay cached in the browser session even after logging out. (RAR-3118)
