OutSystems Developer Cloud 2025-07-02

Released on Jul 02, 2025

New

  • [Mentor AppGen]
    • Expanded Enterprise Integration Capabilities - Mentor-generated apps can now write data directly to public entities from any application of the tenant, making it easier to build connected, enterprise-ready solutions.
      • Guidelines:
        • Write operation for public entities will be available only when the entity has the createOrUpdate action available
        • Delete records from public entities will be available only when the entity has the delete action available
    • New screen template available: Mentor now suggests a lightweight Pop-up screen template instead of generating the standard List, Edit, and Detail screens when an entity has a low number of attributes and no relationships. This enhancement enables faster and more focused interactions, making it ideal for simple entities that require quick edits.
    • AI-Powered UI Personalization
