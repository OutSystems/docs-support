LifeTime Management Console 11.25.2

Released on Dec 05, 2024

Bug Fixing

  • Fixed an issue that only affected cloud infrastructures, where the feature to request a temporary database user was no longer available. (RPM-5414) Application Lifecycle LifeTime

Known issues

  • The RPM-4754 bug fix caused an unintended issue where modules moved between applications are duplicated in the deployment, leading to publishing failures. The workaround is to re-run the deployment plan. To mitigate the issue, the RPM-4754 fix was disabled for OutSystems Cloud environments but still affects self-managed installations.
