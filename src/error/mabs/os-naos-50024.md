# OS-NAOS-50024

## Error message

`An internal error has occurred while fetching mobile configurations`

## Cause

Due to an internal error, ODC cannot retrieve the mobile configurations. Therefore, the mobile package creation failed.

## Impact

Failed to create mobile package.

## Recommended action

Cancel the current package creation attempt, start a new package creation attempt and re-apply the mobile configurations.  

If the problem persists, create a case with [OutSystems support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-NAOS-50024).
