OutSystems Developer Cloud 2026-03-16

Released on Mar 16, 2026

New

  • Workflow Editor: Centralized Error Management
    • You can now manage all the errors of a workflows from a notification panel in Workflow Editor.
    • When a workflow has errors the notification icon will show a counter with the number of errors.
    • Click on that icon and a notification panel is open, where you can visualize, search and navigate to the errors and fix them easily.
