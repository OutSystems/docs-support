OutSystems Developer Cloud 2023-10-11

Released on Oct 11, 2023

New

  • Added a validation to the 'Run now' option in Timers that won't allow triggering more than one execution of a timer at the same time. (RDLOT-1695)
  • Now, the producers list on the app detail screen is sorted by name. (RDLOT-2166)
  • Now it is possible to download a new native version of ODC Studio, which provides performance improvements for MacOS users that have devices with the new M-series processors. (RDLOT-2273)
  • Improved the ODC Portal experience when configuring a timer's schedule, by allowing users to select more effectively the options for calendar-based schedules (monthly, weekly, and daily). (RDLOT-2302)
  • Updated the user's contextual menu on ODC Portal accessible in the top right corner to include the user's picture, name and email. (RDPP-1163)
  • Improved the behavior when filtering for end-user groups and end-user roles. Now, when filtering for a specific app, there exists a specific filter applied to end-user groups, and a specific filter applied to end-user roles that can be used separately. (RDPP-1185)

Bug Fixing

  • Fixed an issue related to permissions when creating mobile packages. Now, with the “Deploy apps” permission, even if the user doesn’t have the “Edit configurations“ permission, it will be possible to create new mobile packages, using the values already configured for previous packages. (RDLOT-2110)
  • Fixed an issue on the create mobile package page, that was not selecting automatically the MABS version used on the previously created package. (RDLOT-2242)
  • Fixed an issue on the app detail screen that was presenting an empty tab placeholder for users who lack permission to access the configurations tab. (RDLOT-2291)
  • Fixed an issue where some App Templates were not presenting the list of producers, on the app detail screen. (RDLOT-2319)
  • Fixed an issue on ODC Portal that was presenting an empty dropdown for some App Templates on the app detail screen. (RDLOT-2321)
