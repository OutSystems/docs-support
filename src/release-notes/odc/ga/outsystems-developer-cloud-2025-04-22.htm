OutSystems Developer Cloud 2025-04-22

Released on Apr 22, 2025

New

  • [Mentor App Editor] New experience to manage authorizations
    • Users can now manage authorization levels for individual entities through a centralized panel available in the Key Icon in the data section.
    • This enhancement streamlines permission control, making it easier to configure and update access levels from a single location.
    (RDMOR-1981)
  • [Mentor App Editor] Increased the complexity of AI suggestions when adding new data
    • AI suggestions now handle more complex data structures, including accurate many-to-many relationships, improving the modeling experience in the editor
    • Add a new entity or attribute now can add more than one data element to support complex relationships
    (RDMOR-1982)
