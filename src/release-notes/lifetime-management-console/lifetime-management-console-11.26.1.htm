LifeTime Management Console 11.26.1

This is a preliminary version of the document.

New in LifeTime Management Console 11.26.1

  • The 'More Security Settings' link will no longer be visible on the Environment Details screen to users who do not have the required permissions to access the Environment Security options. Suggestion made by Evert . (R11CT-3512)
  • A 'Create New Deployment Plan' button has been added to the 'Applications List' screen to accelerate the creation of new deployment plans, for scenarios where multiple active plans are targeted at a given environment. (R11CT-3839)

Bug Fixing

  • When downloading logs from the Audit Logs page in LifeTime, the exported logs now accurately reflect the user information displayed in the UI. (R11CT-3877)
