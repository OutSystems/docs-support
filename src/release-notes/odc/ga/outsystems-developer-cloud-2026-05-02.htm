OutSystems Developer Cloud 2026-05-02

Released on May 02, 2026

Bug Fixing

  • Reverted the fix introduced for NullTextIdentifier() in Aggregate filters. While the previous fix aligned with documented behavior, it introduced an unintended breaking change for some scenarios. The original behavior is restored - NullTextIdentifier() is once again serialized as 'null' in the generated SQL query when used in Aggregate filter expressions. (RPLAT-3865)
