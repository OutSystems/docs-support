# OS-MABS-PRJ-50001

## Error message

`The project configuration is invalid. If the problem persists, open a support case.`

## Cause

This is an internal error. An internal project configuration file has invalid content.

## Impact

The app package can't be generated.

## Recommended action

Regenerate the app package. This might be a temporary issue that resolves on its own.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MABS-PRJ-50001).
