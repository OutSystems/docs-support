ODC Studio 1.7.9

Released on May 25, 2026

New in ODC Studio 1.7.9

  • Updating an OutSystems UI reference from a version earlier than 2.28.0 now triggers an application reload to apply an upgrader that updates Phosphor Icon-related CSS classes. Warning messages are also displayed to help users to fulfil the requirements for using Phosphor Icons as Icon Library in OutSystems UI based themes.

    (RDEV-9465)

  • You can now copy a Mentor response to your clipboard with one click.

    (RDEV-9488)

Bug Fixing

  • Fixed an issue where content copied in the editor could not be pasted back into the application.

    (RDEV-9657)
  • Fixed an issue when the user tried to create a Multilingual Locale in an Agentic app. (RPM-6802) Service Studio
  • Fixed an issue on macOS where repeated keychain access prompts could appear after a previous prompt was denied. (RPM-6843) Service Studio
