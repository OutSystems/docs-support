ODC Studio 1.5.15

This is a preliminary version of the document.

New in ODC Studio 1.5.15

  • The "Remove unused elements" operation is now cancellable. (RDEV-7926)
  • Now it's possible to build complex queries for external entities using SQL Queries in ODC Studio. (RDEV-8407)

Bug Fixing

  • Fixed an issue that caused an unexpected error when the user worked simultaneously in multiple sessions of Studio. (RDEV-8355)
  • Resolved unexpected errors that occurred intermittently to improve overall stability and user experience. (RDEV-8388)
