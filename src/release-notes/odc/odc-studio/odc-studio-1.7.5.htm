ODC Studio 1.7.5

Released on Apr 27, 2026

New in ODC Studio 1.7.5

  • You can now stop ongoing requests in Chat with Mentor by pressing the ESC key or clicking the Stop button. (RDEV-9011)
  • The View Changes section in Mentor now displays 'After mentor changes' instead of 'Current version' to better reflect the changes state. (RDEV-9335)

Bug Fixing

  • Fixed an issue preventing Mentor from setting arguments when calling Actions. (RDEV-9478)
  • Fixed an issue where publish with comment would stop after an automatic merge. (RDEV-9481)
  • Fixed an issue that caused an unexpected error when adding a theme in the mobile app or library. (RDEV-9485)
