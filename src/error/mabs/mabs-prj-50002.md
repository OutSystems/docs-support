# OS-MABS-PRJ-50002

## Error message

`An error occurred while generating the splash screen images. If the problem persists, open a support case.`

## Cause

This is an internal error. The branding image couldn't be transformed or resized to match the iOS or Android splash screen requirements.

## Impact

The app package can't be generated.

## Recommended action

Regenerate the app package. This might be a temporary issue that resolves on its own.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MABS-PRJ-50002).
