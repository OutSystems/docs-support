  • Workflow Editor: Comments
    • In the Workflow Editor, it's now possible to add comments to an activity.
    • Adding a comment can be done in the "..." menu of each activity.
    • It's possible to add comments to all the activity types, except End, Terminate and Go to a previous node .
    • The workflow editor toolbar, has a new icon for comments, by clicking on it, it's possible to view all existent comments of that workflow.
    • After at least one comment is created for an activity, users can add more comments or delete them in the activity's sidepane.
    (RDODCT-3122)
  • Workflow Editor: Copy and paste
    • In the Workflow Editor, it's now possible to copy and paste an activity.
    • It's possible to copy and paste all the activity types, except Start, Conditional Start, End, Terminate and Go to a previous node .
    • The copy can be done in the "..." menu of the activity.
    • The paste can be done in the add flow step menu, appearing as the last option.
    (RDODCT-3123)
