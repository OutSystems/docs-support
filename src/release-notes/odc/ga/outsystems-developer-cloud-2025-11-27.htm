OutSystems Developer Cloud 2025-11-27

Released on Nov 27, 2025

New

  • [Mentor AppGen]
    AppGen now generates field-level validation rules for common scenarios, including mandatory fields, past/future date checks, value range enforcement (Numeric, Text, Phone Number), and email validation, based on your app context or explicit requirements.

    Mentor can now include Pop-up and Tab patterns when generating new screens, providing more flexible and modern UI designs.

    We've implemented Popups ViewMode, allowing you to use Edit and View screens within pop-up modals for a faster, more focused user experience.

    Added support for more dashboard aggregations, including Minimum, Maximum, Average, and Sum, providing richer data visualization options out of the box. (RDMAG-876)

Bug Fixing

  • [Mentor AppGen]

    App Generation crashes:
    <ul>
    <li> Fixed issues where app generation failed when target entities were using reserved system words.</li>
    <li> Resolved a bug causing app generation to fail when multiple stateflows were included in a single screen </li>
    </ul>

    Fixed Sample data Inconsistencies:
    <ul>
    <li>Fixed an issue where sample data was missing for some generated entities.</li>
    </ul> (RDMAG-879)
