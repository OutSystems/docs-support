OutSystems Developer Cloud 2026-02-21

Released on Feb 21, 2026

New

  • We enhanced the publish logic so that it no longer suggests creating a new native mobile build if there is already a native build in progress. (RNMT-7759)

Bug Fixing

  • Fixed issues where updating an extensibility setting or mobile configuration with an empty value produces an unclear error messages. This occurred when updating secret or binary extensibility settings with empty values, or when creating mobile packages with empty certificate, provisioning profile, or keystore files. The system now provides clear error messages indicating that values and files cannot be empty in these contexts. (RNMT-7662)
