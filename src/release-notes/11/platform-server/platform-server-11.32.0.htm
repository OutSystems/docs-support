Platform Server 11.32.0

Released on Dec 02, 2024

New in Platform Server 11.32.0

  • Upgraded NLog libraries to version 5.3.4. (R11BRT-1031)
  • Upgraded Microsoft.AspNet.WebApi.WebHost libraries to version 5.3.0 and Microsoft.AspNet.WebApi.Client to 6.0.0 (R11BRT-1032)
  • Upgraded MySQL driver to version 9.0.0 and Google Protobuf to version 3.26.1. Due to this, MySQL 5.7 or lower is no longer supported as external databases. (R11BRT-1039)
  • Updated System.Text.Json library bundled with the platform from version 8.0.4 to version 8.0.5. This fixes several security vulnerabilities reported in this dependency. (R11BRT-1409)
  • Introduces a new feature that allows specifying in which HTML element of the Dropdown widget an attribute should be rendered. A Dropdown widget is composed of a SELECT element enclosed in a DIV. By default, attributes defined in the widget properties are rendered in the enclosing DIV. This feature allows rendering the attributes in the SELECT element instead. This is limited to Dropdowns with Options Content set to "Text Only" in Reactive and Mobile apps. More information at Dropdown documentation page. (R11DT-2641)
  • Valid from Dec 12th 2024 for any Cloud Platform Server version that supports Log Streaming: For RequestEvent logs streamed through Log Streaming feature, new attributes are supported to enrich troubleshooting and performance analysis use cases (D,EC,LT,SAT,TCIE,TCIT,TEE,TET,TQE,TQT). Learn more at https://success.outsystems.com/documentation/11/monitoring_and_troubleshooting_apps/introduction_to_log_streaming/logged_data_fields/#requestevent-logs (RDUOPT-3640)

Bug Fixing

  • Fixed incorrect database version deprecation notice appearing in Configuration Tool and Database Connections when using a SQL Server 2014 and Azure SQL databases. (R11BRT-1456)
  • Fixed the filename of the exported log file from Service Center for the screen requests and traditional web request logs. Reported by Jo√£o Martins Pereira. (R11CT-3395)
  • Fixed an issue on the error page where the default error message appeared on the first render instead of the custom-defined one. Now, when a custom error message is set, the default error message no longer appears initially. This applies to Mobile applications only. (RPM-4231) Application Runtime Data Access and Manipulation
  • Fixed an issue that prevented the source code of an application from being downloaded correctly, using the LifeTime API /lifetimeapi/rest/v2/applications/{ApplicationKey}/versions/{VersionKey}/content/, when the application includes an extension. (RPM-5136) Application Lifecycle LifeTime
  • Fixed a concurrency issue that caused the publish of an application with processes that have Conditional Start(s) to fail with "The INSERT statement conflicted with the FOREIGN KEY constraint "OSFRK_OSSYS_BPM_ACTIVITY_OSSYS_BPM_PROCESS_PROCESS_ID" while that same application is creating new processes. This only occurred for OutSystems Platforms running SQL Server as the platform database. (RPM-5205) Application Runtime Processes
  • Fixed a security weakness related to Improper Neutralization of Special Elements Used in a Template Engine. (RPM-5239) Application Runtime
  • Fixed an issue where the tenant page in Service Center will present multi-tenant site properties from other tenants that share the same modules with tenant being viewed in a factory with more than 900 modules. (RPM-5257) Application Lifecycle Server.API
  • Fixed an issue causing serial number for EC2 machines using EC2 Launch v2 not being calculated correctly. It may be necessary to retrieve a new license for the correct serial after the upgrade. (RPM-5259) Infrastructure Management Licensing Infrastructure
  • Fixed an issue that caused modules not being able to be debugged when there were inconsistencies with the way key of Application was stored in system entities. (RPM-5271) Application Runtime Debugger
  • Fixed a performance degradation issue on Service Center, affecting only frontends that aren't the Controller. The performance degradation affected specially the Home, Module List and Solution Edit pages and was proportional to the number of modules in the environment. (RPM-5298) Application Lifecycle Service Center
  • Removed dependency to Portable.BouncyCastle that was being introduced by MySql, due to the driver upgrade. This fixes several security vulnerabilities reported in this dependency. CVSSv3.1 score 5.9 (Medium) Customers using custom extensions that rely on Portable.BouncyCastle (BouncyCastle.Crypto.dll) may still be affected. (RPM-5178) Application Runtime
  • Fixed 5 potential security issues related with Improper Input Validation, Authorization Bypass Through User-Controlled Key, Authorization Bypass, Access Restriction Bypass and Open Redirect. Respective CVSSv3.1 scores 8.1, 6.2 , 6.4 , 6.4 , 5.3. (RPM-5207) Application Runtime Logic Execution
  • Fixed a potential security issue related to Regular Expression Denial of Service (ReDoS). CVSSv3.1 score 7.5 (High). (RPM-5214) Application Runtime
  • Fixed a potential security issue related to Prototype Pollution attacks. CVSSv3.1 score 7.3 (High). (RPM-5234) Application Runtime
