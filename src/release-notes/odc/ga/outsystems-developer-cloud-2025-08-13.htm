OutSystems Developer Cloud 2025-08-13

Released on Aug 13, 2025

New

  • Access to Add-On-related features in the ODC Portal is now controlled by the subscription status of their respective Add-Ons:
    • IP Filters is now available only with an active Sentry subscription.
    • Private Gateways is now available only with an active Private Gateways subscription.
    (RDLOT-3873)
  • Workflow Editor: Enhanced Search Capabilities
    • The Workflow Editor now includes advanced search functionality to help you quickly find what you need.
    • Click on the magnifying glass icon in the top toolbar, next to the Publish button, to open a modal with powerful search options.
    • You can also use the keyboard shortcuts Ctrl+K (Windows) or Cmd+K (Mac) to open the search modal.
    • You can search flow steps, actions, expressions, commands, and more, making it easier to navigate complex workflows.
    (RDODCT-3418)
  • A six-digit format will now be used for self-registration codes. (RDPIB-2789)
