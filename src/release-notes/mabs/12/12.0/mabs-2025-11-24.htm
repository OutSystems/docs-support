MABS 2025-11-24

Released on Nov 24, 2025

New

  • Added the inset injector plugin to our native template to ensure all apps built with the template have the new CSS variables available. (RDMR-1043)
  • Fixed a problem causing the Android navigation bar to have a solid background color when preference AndroidEdgeToEdge is set to true and running on Android 15 (RDMR-1074)
  • Added support for Android Predictive Back Button on OutSystems cordova applications (RDMR-964)

Bug Fixing

  • Fixed a problem preventing DisplayName preference to successfully have an effect (RDMR-1079)
  • Fixes issues with single sign on flow when Chrome isn't the default browser, allowing users to log in using any browser in Android. (RPM-5935)
