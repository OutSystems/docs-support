OutSystems Developer Cloud 2025-06-02

Released on Jun 02, 2025

New

  • Now, the role of the user opening a workflow activity will be displayed, if the workflow is assigned to that role (RDASS-2345)
  • For SQL Server, Azure SQL, and Oracle integrations through Private Gateway, we've simplified the configuration process. You can now easily enable the integration by toggling the new Private Gateway option and specifying the Private Gateway port used in your Cloud Connector setup. (RDCIST-3004)
  • For PostgreSQL integrations through Private Gateway, we've simplified the configuration process. You can now easily enable the integration by toggling the new Private Gateway option and specifying the Private Gateway port used in your Cloud Connector setup. (RDCIST-3006)
  • The form in the ODC Portal now accepts the domain search.azure.us for Azure AI connections. (RDCIST-3108)
