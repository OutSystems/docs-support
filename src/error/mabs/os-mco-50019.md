# OS-MCO-50019

## Error message

`Extensibility Configurations of <Asset> are not valid for AppShield. If you didn't change the asset, open a support case.`

## Cause

An internal error occurred.

## Impact

The mobile compilation fails, causing the native mobile build to fail.

## Recommended action

Check that you haven't modified the extensibility configurations of the asset mentioned in the error message.

Please try the operation again. This might be a temporary issue that resolves on its own.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MCO-50019).
