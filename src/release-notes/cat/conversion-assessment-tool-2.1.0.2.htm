Conversion Assessment Tool 2.1.0.2

This is a preliminary version of the document.

New in Conversion Assessment Tool 2.1.0.2

  • Added a new data warning that flags modules and user providers that multiple tenants associated. (RADRT-4189)
  • You can now drag plans within the dependencies diagram to customize your view. The diagram auto-saves your layout changes, and you can reset it to the default view at any time. (RADRT-4192)
  • Added a new data warning to notify you when custom user providers are being used. (RADRT-4197)
  • You can now manage plans directly from the graph, using the new context menu available in the plan dependencies graph view. (RADRT-4226)

Bug Fixing

  • Fixed a bug that was blocking data assessment when using Multiple Database Catalogs. (RADRT-4258)
  • Fixed an issue in data assessment that prevented recently created assets from appearing on the ODC Blueprint screen. (RADRT-4259)
