OutSystems Developer Cloud 2025-06-12

Released on Jun 12, 2025

New

  • Enabled HTTP Strict Transport Security (HSTS) for all requests, to enforce secure HTTPS connections. (RDGRS-1354)

Bug Fixing

  • Fix issue where raising an exception with a value other than text would cause a runtime error (RAR-3460)
  • Fixed a memory leak and bug where block and screen spans wouldn't be dispatched when there was an exception thrown on the OnReady or OnRender event handlers. (RAR-3463)
