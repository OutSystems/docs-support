ODC Studio 1.5.12

This is a preliminary version of the document.

New in ODC Studio 1.5.12

  • The "Checking for dependency updates" operation is now cancellable. (RDEV-8282)

Bug Fixing

  • Fixed an issue where a message was not displayed when the source of the aggregate is a local entity. (RDEV-7835)
  • Fixed an error message in aggregates to enhance clarity by removing the 'above' keyword. (RDEV-8330)
  • Fixed an issue that prevented opening a screen or flow while "Edit app properties" dialog is opened. (RDEV-8348)
  • Fixed an issue that would cause an unexpected error when scaffolding an entity. (RPM-5552)
  • Fixed an issue that caused the "Publish" button not to change to "Open in browser" after publishing. (RPM-5632)
