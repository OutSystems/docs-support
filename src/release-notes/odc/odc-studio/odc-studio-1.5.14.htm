ODC Studio 1.5.14

This is a preliminary version of the document.

New in ODC Studio 1.5.14

  • You will now see a pop-up dialog explaining why authentication is required when previewing an app without assigned end-user roles. (RDEV-8347)

Bug Fixing

  • Fixed an issue where context menus could close unexpectedly when hovering over disabled menu items with the pointer. (RDEV-8351)
  • Fixed an issue where MacOS users lost their login credentials after restarting their computer. (RDEV-8353)
  • Fixed an issue where ODC Studio occasionally froze and failed to publish an application, throwing an unexpected error. (RDEV-8379)
  • Fixed an issue where changing a property in certain widgets, such as the Dropdown, could cause an error. (RPM-5611)
