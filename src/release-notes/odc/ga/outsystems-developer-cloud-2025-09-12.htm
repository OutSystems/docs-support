OutSystems Developer Cloud 2025-09-12

Released on Sep 12, 2025

New

  • This version introduces a new Architecture pattern to Code Quality: "Application Referencing REST Service directly without using a Library" (RDASS-3331)
  • Workflow Editor: Revisions History
    • The Workflow Editor now includes Revision History feature.
    • Access it from the top toolbar by clicking the "..." menu and selecting Open revisions history.
    • A side pane will open, showing a timeline of all revisions created for that workflow. You can navigate through them.
    • You can select any version to view it in read-only mode inside the editor.
    • You can also revert to a previous version and publish it if needed.
    (RDODCT-3487)
