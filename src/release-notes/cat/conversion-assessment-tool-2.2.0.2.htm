Conversion Assessment Tool 2.2.0.2

Released on Apr 15, 2026

New in Conversion Assessment Tool 2.2.0.2

  • Added a new rule to flag Site Properties which are being changed programmatically. (RADRT-3739)
  • Added support for deleting Probe Connections to streamline environment configuration. (RADRT-4306)
  • Added support for LDAP authentication to streamline user login and access management. (RADRT-4309)
  • Added support for using Service Center as a fallback login provider when using an unsupported external authentication method. (RADRT-4310)
