MABS 2023-10-26

Released on Oct 26, 2023

New

  • MABS 10 now uses Node 18 and NPM 8. (RNMT-6064)
  • Update Native Shell to target the latest iOS 17 SDK. (RNMT-6167)
  • Update Native Shell to target the latest Android 14 SDK. (RNMT-6192)

Bug Fixing

  • Fixed an issue that caused the build to fail in some edge cases, when using the cordova preferences android-minSdkVersion and deployment-target. (RNMT-6290)
  • MABS 10 now uses SQLite version 3.40.0. (RPM-4466) Mobile Apps Builder

Breaking Changes

  • The legacy Android Support Libraries (com.android.support) are no longer supported. Builds containing these libraries will fail until they are migrated to their respective AndroidX counterpart. Owners of non-supported plugins using Android Support Libraries must migrate their plugins to work with MABS 10. (RNMT-6187)
  • Removed functionality that was specifically adding aps-environments to plist when present in the provisioning profile. This should be handled by each plugin making use of the aps-environment entitlement, and not generically by MABS since not all applications need this. (RNMT-5935)
