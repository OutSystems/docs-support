OutSystems Developer Cloud 2025-01-06

Released on Jan 06, 2025

New

  • On the Workflow logs it is now possible to see the Instance Label for each log, on the workflows that defined it (RDASS-2403)
  • You can now search workflow instances by their ID or Label (when defined) in the Workflows detail screen. If the instance has a label defined, it will also be displayed below the Instance ID. (RDLOT-3354)

Bug Fixing

  • Fixed an issue on the ODC Portal where subscription usage percentage displayed decimal values. (RDLOT-3351)
