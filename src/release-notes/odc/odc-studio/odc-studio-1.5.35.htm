ODC Studio 1.5.35

Released on Aug 18, 2025

Bug Fixing

  • Fixed an issue that causes Actions with input parameter of Entity type to have outdated Source parameters, namely after changes to the Action being executed, or the 'Is AutoNumbered' property of the Id Attribute. (RDEV-8703)
  • Fixed an issue where the debugger would incorrectly display variable values wrapped in XML tags instead of plain text. (RDEV-8709)
