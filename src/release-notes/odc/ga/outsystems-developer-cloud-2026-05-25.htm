OutSystems Developer Cloud 2026-05-25

Released on May 25, 2026

New

  • Manage end-user groups permission now includes group mapping management: users with this permission can view, create, edit, and delete group mappings directly from the end-user group details page. Previously, this required granting the broader Manage authentication permission. This change enables finer-grained control over managing end-user group mappings.

    (RDPP-3029)
