OutSystems Developer Cloud 2024-10-28

Released on Oct 28, 2024

Bug Fixing

  • Fix for a bug where when a screen has a Long or Decimal input, in some specific scenarios, the OnParametersChange would not trigger (RAR-2950)
  • Fixed an issue with Salesforce integrations where metadata and data were not refreshing after editing the connection to point to a different Salesforce instance. Previously, the displayed information remained linked to the previous instance. (RDCIST-2454)
