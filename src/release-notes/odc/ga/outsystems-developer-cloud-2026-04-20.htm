OutSystems Developer Cloud 2026-04-20

Released on Apr 20, 2026

New

  • When creating iOS mobile app packages in the ODC Portal, you can now select the Simulator build type, allowing you to run and test your app in the Mac iOS simulator without the need of certificates and profiles. (RDLOT-4388)

Bug Fixing

  • Fixed an issue where Log entries for Timer executions were missing in ODC Portal. This occurred when the Timer logic failed on its last retry (fourth attempt). Timer execution Logs are now visible as expected. (RTAFB-9105)
