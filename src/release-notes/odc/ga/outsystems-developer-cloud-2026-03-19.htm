OutSystems Developer Cloud 2026-03-19

Released on Mar 19, 2026

New

  • Workflow Editor: Open documentation from node
    • In the Workflow Editor, it's now possible to open the documentation related with a specific activity type.
    • In the "..." menu of each activity you can see an option "Help with".
    • Click on that option and the documentation related with that activity will be open in a new tab.
    (RDODCT-4087)

Bug Fixing

  • Fixed an issue that caused Aggregate filters using NullTextIdentifier() to return no results, even when records with empty Text Identifier attributes existed. This occurred when NullTextIdentifier() was used directly in an Aggregate filter expression, where it was incorrectly serialized as null in the generated SQL query instead of "" (empty string). Now, NullTextIdentifier() is correctly evaluated as an empty string in Aggregate filters, consistent with the documented behavior where NullTextIdentifier() = "" returns true. (RPM-6423)
