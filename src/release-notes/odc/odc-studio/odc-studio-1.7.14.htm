ODC Studio 1.7.14

Released on Jul 07, 2026

New in ODC Studio 1.7.14

  • Discarded plans in Mentor now appear with their title struck through and with a "Discarded" label, making it clear which proposed changes are no longer active.

    (RDEV-9647)

Bug Fixing

  • Fixed an issue on Windows where pasting from the clipboard history (Win+V) into the expression editor, the advanced SQL editor or description fields, always inserted the most recent entry instead of the one selected from the history.

    (RDEV-9865)

  • Fixed an issue where the scroll to bottom button in the Mentor panel was not visible in light theme.

    (RDEV-9867)

  • Fixed an issue where sometimes the styles tab for a given widget would be empty.

    (RDEV-9871)

  • Fixed an issue where the scroll on the collapsible pane in error messages was not visible.

    (RDEV-9906)

  • Fixed an issue where a color picker was incorrectly displayed in SQL statement editors and expression editors.

    (RDEV-9907)
  • Fixed an issue causing the enabled state of nodes to be lost during merges. (RPM-6911) Service Studio Merge operation
