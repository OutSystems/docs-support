# OS-MCO-50008

## Error message

`An error occurred while listing the versions of a package. If the problem persists, open a support case.`

## Cause

An internal error occurred.

## Impact

The mobile compilation fails, causing the native mobile build to fail.

## Recommended action

Check that you haven't modified the extensibility configurations of any supported OutSystems library or plugin.

Please try the operation again. This might be a temporary issue that resolves on its own.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MCO-50008).
