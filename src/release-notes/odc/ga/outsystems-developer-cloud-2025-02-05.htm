OutSystems Developer Cloud 2025-02-05

Released on Feb 05, 2025

New

  • For SAP OData connections, attributes that are generated but not insertable, updatable, or deletable will no longer be displayed in the ODC Portal. (RDDFC-3623)

Bug Fixing

  • Fixed an issue that caused the User Entity from the System producer not to be pickable when trying to add it in the preview step of Mentor. Now, the User Entity can be picked and used to generate applications with Mentor. (RDMAG-594)
  • Fixed an issue that caused Mentor to fail when generating applications. This occurred when the user provided prompts and/or requirement documents that included characters with diacritics. Now, Mentor is more robust and is able to generate applications even when these characters are present. (RDMAG-608)
