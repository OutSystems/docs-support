Platform Server 11.41.1

This is a preliminary version of the document.

New in Platform Server 11.41.1

  • Added support for the form-action and report-to Content Security Policy (CSP) directives. The platform now sends the modern Reporting-Endpoints HTTP header, used by report-to, alongside the legacy report-uri directive for CSP violation reporting, providing improved browser compatibility with the latest CSP specification.

    (R11DT-3597)

Bug Fixing

  • Fixed an issue that caused process events to accumulate indefinitely after processing failures, even after the configured backoff period had elapsed. This occurred because failed events were requeued without checking if they had already exceeded the backoff window, preventing old events from ever being cleaned up. Now, before requeuing a failed event, the platform checks whether the event has been waiting longer than the allowed backoff period and deletes it instead of requeuing it. Normal events are removed after 7 days, and light events after 30 days. (RPM-4745) Application Runtime
  • Added support for more SAP date/time formats, including the 24:00:00 midnight convention. You can configure these formats in Factory Configuration > Platform Configurations > Runtime > SAP DateTime Format (RPM-4905) Application Runtime Logic Execution
  •  Fixed an issue that could cause user registration failures when usernames exceeded 50 characters (RPM-4932) Application Lifecycle Service Center
  • Fix issue where deleting entity attributes did not trigger the refresh warning indicating structural changes in Exposed Entity on Lifetime. (RPM-6379) Application Lifecycle Server.API
  • Fixed an issue that caused the Feedback_AjaxWait component to fail or trigger console errors. This occurred during rapid asynchronous requests where the feedback message state was not properly synchronized with the AJAX lifecycle. (RPM-6398) Application Runtime System Components
  • Fix CleanLoginAttempts Timer from Users runs multiple times due to the fact it's incorrectly marked as Multi-Tenant. (RPM-6608) Application Runtime Authentication and Authorization
  • Fixed an issue that caused deployment of Static Entities with Multilingual be aborted due to missing index in Oracle Databases when deployed to a different database schema other than Main. (RPM-6618) Publish Operation Deployment stage
