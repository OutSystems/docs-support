OutSystems Developer Cloud 2025-07-08

Released on Jul 08, 2025

New

  • In the ODC Portal, it is now possible to access a comprehensive and chronological record of actions across ODC Portal, Studio, and APIs through the new Audit Trail Console. Easily search for specific events through an intuitive interface, or export detailed logs in CSV format for in-depth analysis and long-term retention. (RDASS-3056)
  • The ODC Portal now allows to configure the streaming of observability data - application logs, traces, and metrics - directly to third-party application performance monitoring tools (e.g., Dynatrace, New Relic) or cloud storage (e.g., AWS S3) in OpenTelemetry format, through the App Analytics Stream console, on the Configure section (RDASS-3057)
