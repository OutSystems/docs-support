ODC Studio 1.7.6

Released on May 04, 2026

New in ODC Studio 1.7.6

  • You can now see a clear message in Mentor when Fair Usage limits are reached. (RDEV-8990)
  • Mentor can now handle Agentic Apps and Libraries as part of a new beta experience. (RDEV-9568)

Bug Fixing

  • Improve merge reliability on large assets. (RDEV-9484)
  • Fixed an issue that would cause temporary unresponsiveness when launching ODC Studio for the first time. (RDEV-9486)
  • Improved the detection of duplicate static records when the Identifier attribute is of type Text. (RDEV-9521)
