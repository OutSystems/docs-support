ODC Studio 1.5.0

Released on Dec 16, 2024

New in ODC Studio 1.5.0

  • It is now possible to right-click on an Entity or Entity Action to instantly create wrappers with basic validations. (RDEV-8087)
  • It is now possible to filter by author and revision in "Open other revision". (RDEV-8121)

Bug Fixing

  • Fixed an issue that could cause the application to become unresponsive when interacting with toolbar options. (RDEV-8118)
  • Fixed an unexpected error when publishing an asset with a public role within a folder. (RDMST-3347)
