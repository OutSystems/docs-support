OutSystems Developer Cloud 2025-04-30

Released on Apr 30, 2025

New

  • Monthly Security Update (April 2025): Updated application container base image, including security fixes. Check Microsoft .NET 8.0.15/8.0.15 release for more details: https://github.com/dotnet/core/blob/main/release-notes/8.0/8.0.15/8.0.15.md (RRCT-6372)

Bug Fixing

  • Using a Data Action output variable in its own On After Fetch event handler no longer causes an error. (RAR-3319)
  • Using a block id in an input parameter of an On After Fetch handler no longer causes an error. (RAR-3336)
  • Fixed an issue that caused the builtin and external login processes to become stuck when the end-user's system clock was ahead of the real time by at least 5 minutes and no more than 12 hours. Now, the login process will fail and an error will be logged informing that the user's system clock might be wrong. (RPM-5787)
  • Fix sanitization of formulae when generating Excel files using the RecordListToExcel tool. Now, the single quotation mark is no longer part of the value. Instead, the cell is marked to be quoted internally which hides the visible quotation mark from being displayed. (RRCT-6335)
