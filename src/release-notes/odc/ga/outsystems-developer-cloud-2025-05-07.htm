OutSystems Developer Cloud 2025-05-07

Released on May 07, 2025

New

  • [Mentor AppGen]
    • Performance Improvements: Reduced app generation average time by parallelizing app data and sample data generation processes
    • UI Diversity Enhancements: Introduced a new detail screen layout, cardPerson, which becomes available when the entity contains attributes typically associated with a person (e.g., First Name, Last Name, and Picture)
    • Improved Requirement Processing - AppGen now handles requirement documents and prompts that involve a higher number of entities more effectively, ensuring better generation accuracy for complex app specifications
    (RDMAG-749)
  • [Mentor App Editor]
    • AI Data Suggestions Enhancements: AI suggestions can now intelligently mix both local and external entities when proposing data to be added.
    (RDMOR-2023)

Bug Fixing

  • [AppGen] Fixed a crash in the OML generator when the User entity had a self-referencing ID (RDMAG-740)
