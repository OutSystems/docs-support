OutSystems Developer Cloud 2024-03-07

Released on Mar 07, 2024

New

  • You can now configure a default domain for each stage from the Domains section of the ODC Portal. This allows you to use that domain programmatically in your OutSystems apps, for example, when sending emails that contain links to your apps. For more information, refer to our documentation. (RDEL-2139)

Bug Fixing

  • Empty List widgets without any scroll can no longer crash the screen rendering. (RAR-2543)
