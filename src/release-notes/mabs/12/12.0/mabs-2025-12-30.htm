MABS 2025-12-30

Released on Dec 30, 2025

New

  • Users are no longer prompted to open the inspector with a three-finger press on iOS Development builds. (RDMR-1036)
  • Adopt capacitor 8.0.0 for OutSystems capacitor applications (RDMR-1055)
  • Added support for predictive back gesture on OutSystems Capacitor Android applications (RDMR-1118)

Bug Fixing

  • Fixed handling of IME (e.g. virtual keyboard) on OutSystems cordova applications in Android (RDMR-1126)
  • Fixed a problem with deeplink handling on OutSystems iOS capacitor applications when the deeplink URL contained mixed text case characters (RDMR-1148)
  • Resolved a failure in Over-the-Air (OTA) updates occurring during framework migrations between Cordova and Capacitor. (RDMR-1150)
  • Fixed a problem where, randomly, applicationId wouldn't be properly set for OutSystems Capacitor Android applications (RDMR-1161)
