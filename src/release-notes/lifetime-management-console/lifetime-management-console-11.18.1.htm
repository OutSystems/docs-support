LifeTime Management Console 11.18.1

Released on Oct 18, 2023

New in LifeTime Management Console 11.18.1

  • Platform Server version updated to 11.23.0. (R11CT-2436)

Bug Fixing

  • Fixed an issue with displaying staging records when the advanced filter is expanded in the Stagings List screen. This issue would occur when the user expands the advanced filter causing the first record to be partially hidden at the top. Now, the first record is displayed correctly regardless of the advanced filters being expanded or collapsed. (R11CT-2474)
  • Fixed an issue in which SQL Server 2019 was not being included in the list of database technologies of LifeTimeâ€™s checklist. (R11CT-2486)
  • When configuring OpenID Connect providers, LifeTime now supports longer client secrets - up to 100 characters in length, from the previous limit of 50 characters, ensuring the identity providers generating longer client secrets can be configured seamlessly. (RPM-4050) Application Lifecycle LifeTime
  • Fixed an issue that caused excessive CPU usage. This occurred when the Staging Edit page was open. (RPM-4316) Application Lifecycle LifeTime
  • Fixed an issue that would cause 2-stage deployment plans to finish with errors. This issue occurred when 'Continue Deployment' was clicked after the solution associated with the plan was deleted in the target environment during the execution of the SolutionPack_Cleanup timer. Now, LifeTime will receive the information that the solution was deleted and will abort the deployment plan. (RPM-969) Application Lifecycle LifeTime

Note

  • In this LifeTime version, manual installation of Microsoft Visual C++ 2015-2022 is required. The checklist bundled with the installer doesn't include the necessary steps for that. We acknowledge that this is a known issue, please download the checklist from our website that includes the steps for the manual installation of Microsoft Visual C++ 2015-2022.
