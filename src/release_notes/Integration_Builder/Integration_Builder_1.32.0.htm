New in Integration Builder 1.32.0

  • Improved the layout of the "Review" screen for Dataverse, Dynamics, Salesforce, SAP OData, and Sharepoint integrations. (RCNT-4278)
  • Added a visual indicator that table rows can be clicked on the "Add tables" step of the integration wizard for relational databases. (RCNT-4576)
  • A MongoDB integration is now available, as a Technical Preview. Easily connect your apps with the most popular no-SQL database, MongoDB. The new integrations are available exclusively in the latest version of Integration Builder and fully supported by the OutSystems platform. Find out more about MongoDB integrations in our documentation. (https://www.outsystems.com/goto/ib-mongodb-docs) (RCNT-4673)

Bug Fixing

  • Fixed a generation error when object names get duplicated due to special characters. (RCNT-4602)
  • Improvement on the "Add tables" step of the relational database integration wizard. Now long table names don't cause a horizontal scroll. (RCNT-4662)
  • Fixed: Integration Builder fails to generate Salesforce connectors when object names don't contain roman alphabet characters (RPM-2266)
