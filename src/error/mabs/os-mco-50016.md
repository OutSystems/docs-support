# OS-MCO-50016

## Error message

`The template NPM package is invalid. If the problem persists, open a support case.`

## Cause

An internal error occurred.

## Impact

The mobile compilation fails, causing the native mobile build to fail.

## Recommended action

Please try the operation again. This might be a temporary issue that resolves on its own.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MCO-50016).
