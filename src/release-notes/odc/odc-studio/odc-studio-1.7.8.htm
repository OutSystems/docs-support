ODC Studio 1.7.8

Released on May 18, 2026

New in ODC Studio 1.7.8

  • Mentor now brings your last prompt back to the input area when you stop it.

    (RDEV-9519)

  • Mentor now greets you with a fresh message each time you start a new conversation.

    (RDEV-9580)

  • The App properties dialog now features vertical tabs for easier navigation. Mobile settings are grouped in a Mobile section with tabs for Appearance, Offline sync, and Extensibility. You can search by tab name or properties to quickly find settings.

    (RDMDX-254)

Bug Fixing

  • Updated Studio's SQLite version and fixed CVE 2025-6965.

    (RDEV-9615)

  • Fixed an issue that caused an unexpected error when cloning a protected app.

    (RDEV-9634)
  • Fixed an issue that would prevent users to publish an app due to a block incompatibility with its definition. (RPM-6749)
