Platform Server 11.39.0

Released on Nov 10, 2025

New in Platform Server 11.39.0

  • Added a new capability that allows the expose of views for entities in the database using LifeTime, allowing names to be consistent between environments. This capability will require a future version of LifeTime to use it. (R11BRT-1998)
  • Added support for REDIS 8.2.2 (or higher, within the 8.2 line) as the session database for on-premise customers. (R11BRT-2136)

Bug Fixing

  • Fixed an issue that caused DBCleaner_API's Attribute_ListDeleted to return active attributes which have the same name as one that was previously deleted. (R11BRT-2027)
  • Fixed an issue in Server.API that caused slowness in some endpoints related to Entities. (R11BRT-2037)
  • Fixed an issue that caused false positives in the detection of breaking changes of processes during a publication.
    This occurred when the processes had a loop pattern.
    Created a new algorithm to reduce the number of false positives. This algorithm has a performance impact and can be enabled via Factory Configuration. (RPM-6022)
  • Fixed an issue where Service Center would fail to call the compiler service due to a cached telemetry initialization error. (RPM-6035)
  • Updated a third-party Platform dependency to address a known vulnerability (CVSS 9.1). Detailed analysis confirmed the Platform and Applications are not vulnerable to the reported vulnerability prior to this update. No immediate action is required regarding this vulnerability. (R11DT-3356)
