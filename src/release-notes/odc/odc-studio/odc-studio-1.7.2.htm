ODC Studio 1.7.2

Released on Apr 06, 2026

New in ODC Studio 1.7.2

  • You can now add a comment when publishing a new revision of an app. (RDEV-9287)

Bug Fixing

  • Fixed an issue that could trigger an error while loading icons and images when the Merge Editor was open. (RDEV-9251)
  • Fixed an issue that caused the Data Type picker dialog to open unexpectedly when clicking on a Data Type property in the search results. Now, clicking navigates to the property as expected. (RDEV-9396)
  • Fixed an unexpected error that occurred when Mentor removed specific widgets. (RDEV-9415)
