# OS-MCO-40009

## Error message

`The app includes too many plugins. Reduce the number of plugins.`

## Cause

Your mobile app includes too many plugins, exceeding the maximum allowed limit.

## Impact

The mobile compilation fails, causing the native mobile build to fail.

## Recommended action

Remove some plugins from your mobile app to reduce the total number of plugins.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MCO-40009).
