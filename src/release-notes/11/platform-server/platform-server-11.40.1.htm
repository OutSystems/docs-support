Platform Server 11.40.1

Released on Feb 09, 2026

New in Platform Server 11.40.1

  • To enhance data privacy, REST Expose now automatically redacts Authorization headers in integration logs, ensuring consistency with the existing REST Consume logic. (R11DT-3426)

Bug Fixing

  • Fixed warnings in detached Applications and Scheduler source related to the usage of third party libraries that are not supported in older .Net Framework versions. This change has no runtime impacts, as all versions were already compatible with the current OutSystems Platform system requirements. (R11BRT-2206)
  • Fixed an issue on the Extension log page where sometimes the user wasn't able to see any results if filtering by (All) extensions while using Log Separation. (RPM-4823) Application Lifecycle Service Center
  • Fixed a security vulnerability that could lead to account takeover when using SAML based authentication mechanisms. (RPM-6374) Infrastructure Management Cloud Backbone
