# OS-MABS-PRJ-50003

## Error message

`An error occurred while writing or copying a file. If the problem persists, open a support case.`

## Cause

This is an internal error. An internal file required for the build couldn't be written or copied.

## Impact

The app package can't be generated.

## Recommended action

Regenerate the app package. This might be a temporary issue that resolves on its own.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MABS-PRJ-50003).
