MABS 2025-11-12

Released on Nov 12, 2025

New

  • Breaking change: Cordova based android applications no longer get overlaid by the status bar. Additionally, the cordova-plugin-statusbar plugin is no longer part of OutSystems mobile applications (Android and iOS) (RDMR-903)
  • Fixed a scenario where MABS would output the generic plugin error OS-MABS-PLG-50000 instead of the specific one OS-MABS-PLG-40002. (RNMT-7520)

Bug Fixing

  • Fixed an issue when building Capacitor apps, related to a breaking release by Rollup. (RDMR-1070)
  • Fixed an issue that prevented Capacitor plugins from setting the Kotlin version in Capacitor applications. (RDMR-943)
