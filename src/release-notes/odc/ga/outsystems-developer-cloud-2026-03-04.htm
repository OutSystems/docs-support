OutSystems Developer Cloud 2026-03-04

Released on Mar 04, 2026

Bug Fixing

  • Previously, when using an External IdP for authentication, specific unsupported configurations, such as invalid redirect URIs or expired session tokens, resulted in the display of unstyled system error pages. These pages lacked application branding and navigation, preventing users from returning to the app. Now, these authentication errors redirect to the application's configured error pages, allowing users to retry the login or navigate back to the home screen. (RPM-6316)
