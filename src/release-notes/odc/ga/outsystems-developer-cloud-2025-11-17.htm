OutSystems Developer Cloud 2025-11-17

Released on Nov 17, 2025

New

  • <p><strong>Workflow Editor: Keyboard Shortcuts</strong></p>
    <ul>
        <li>Workflow Editor now supports keyboard shortcuts, significantly improving development speed and productivity on both Windows and macOS operating systems.</li>
        <li>A complete list of all available shortcuts can be quickly accessed by clicking the question mark icon (?) in the top bar and selecting the new <strong>Shortcuts</strong> option.</li>
