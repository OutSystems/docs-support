OutSystems Developer Cloud 2025-05-22

Released on May 22, 2025

New

  • [Mentor AppGen]
    • Improved Requirement Processing: Mentor now more accurately extracts data model and authorization information from requirements documents
    • UI Diversity Enhancements: Introduced a new detail screen layout, List screen with Map View, which becomes available when the entity contains attributes typically associated with an address or location
      • Guidelines: Make sure to configure a valid API key in ODC Studio to display markers on the map
    • Logged-in user authorization use cases expanded: Mentor can create authorizations where users can manage their own data while having limited access to all recordsâ€”for example, allowing the user to view all orders but only editing their own orders
    • AI-Suggested Menu Icons: Mentor now automatically suggests Font Awesome icons for each link in your app menu, enhancing visual clarity and user experience.
    • Create Apps Using Your Own Theme: Mentor now detects the theme name from the prompt or requirements document and applies it to the generated app automatically.
      • Guidelines: Ensure your theme has the layout property of "blocks" filled in and that the chosen layout has the following placeholders: Header, Breadcrumbs, Title, Actions, and MainContent
    (RDMAG-767)
  • [Mentor App Editor]
    • Logged-in user authorization suggestions: AppEditor now provide authorization suggestion rules based on the logged-in user. This accelerator enables scenarios such as allowing the user to view all orders but only editing their own orders
    (RDMOR-2071)

Bug Fixing

  • Fixed an issue where the instance flow wasn't rendering correctly when opening the details of a workflow instance. (RDLOT-3640)
