# OS-MCO-50018

## Error message

`Extensibility Configurations of <Asset> could not be deserialized. If the problem persists, open a support case.`

## Cause

An internal error occurred.

## Impact

The mobile compilation fails, causing the native mobile build to fail.

## Recommended action

This error can be caused by incorrect data types or values in the properties of the extensibility configurations of the asset being used. Correct these issues to potentially resolve the problem.

Please try the operation again. This might be a temporary issue that resolves on its own.

If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-MCO-50018).
