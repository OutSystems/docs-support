Cross-platform Service Studio 11.55.22

This is a preliminary version of the document.

Bug Fixing

  • Fixed an issue that could cause the application to unexpectedly shut down during drag-and-drop operations. (RDEV-8489)
  • Fixed an issue that causes an unexpected error when changing the name of 'View Entity Data' elements to empty. (RDEV-8494)
  • Fixed an issue where clicking the Module menu could cause Service Studio to become unresponsive. (RDEV-8496)
  • Fixed an issue that caused an unexpected error when performing undo and redo operations. (RDEV-8499)
  • Fixed an issue that caused an unexpected error when selecting an element in element pickers, such as 'Add Source' in Aggregates. (RDEV-8500)
