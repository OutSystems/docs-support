Platform Server 11.35.0

Released on May 12, 2025

New in Platform Server 11.35.0

  • Added support for .NET Framework 4.8.1. (R11BRT-1152)
  • Added support for Windows Server 2025. (R11BRT-1524)
  • Build Tools is no longer a mandatory requirement for the installation of Platform Server. Build Tools is only needed when upgrading extension from previous Major Platform versions (like O10). Platform Server installer no longer installs Build Tools automatically. Checklist was updated to reflect this change. (R11BRT-1724)

Bug Fixing

  • Fixed an issue where the App Feedback would not take into consideration the User's Group Control for Reactive Web Apps. Due to its current architecture, App Feedback still presents an issue in the transition between Anonymous and Authenticated. (RPM-3223) Application Lifecycle App Feedback Backoffice / ECT
  • Improved how sessions are registered when using SAML on Mobile Apps. This should prevent log in errors that could otherwise happen in some situations ("data would be truncated"/"value too large for column") (RPM-4965) Application Lifecycle Users
  • Fixed an issue that caused LifeTime to become stuck at "Creating solution pack" when messages obtained were unordered while using Oracle RAC. Due to the necessary changes, there will be an impact in performance to guarantee ordering while using Oracle RAC. (RPM-5372) Application Lifecycle LifeTime
  • Fix a concurrency issue that could cause errors during the Solution Publish operation when, on rare occurrences, the same file was being accessed by more than one thread simultaneously. (RPM-5643) Publish Operation Deployment stage
  • Renamed the authentication option from Azure ID to Microsoft Entra in Users to align with Microsoft's updated branding. (RPM-5706) Application Lifecycle Users
  • Fixed a potential security issue related to mutation cross-site scripting (mXSS). CVSSv3.1 score 4.5 (Moderate). (RPM-5630) Application Runtime
