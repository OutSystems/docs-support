LifeTime Management Console 11.17.3

Released on Aug 09, 2023

Bug Fixing

  • Fixed an issue that would redirect the user to the internal exception screen. This issue would be triggered when the user tried to access a saved staging plan which included an environment they don't have access to. Now, the user can't navigate to staging screens that include environments to which the user doesn't have access. (R11CT-2247)
  • Fixed an issue that would prevent a query from returning all the results for the StagingReport. This issue occurred when trying to export a StagingReport from the Troubleshoot page when there were no Source or Target environments filtered. Now, the page filter no longer has an impact on the report results and all the records are exported. (R11CT-2251)
  • Fixed an issue that showed a hotfix tooltip with the wrong environment name. This issue occurred when a user didn't have access to the first environment on the infrastructure and their first visible or following environments had a hotfix. Now the tooltip doesn't mention the first environment's name if the user doesn't have permissions for it. (RPM-876) Application Lifecycle LifeTime

Note

  • In this LifeTime version, manual installation of Microsoft Visual C++ 2015-2022 is required. The checklist bundled with the installer doesn't include the necessary steps for that. We acknowledge that this is a known issue, please download the checklist from our website that includes the steps for the manual installation of Microsoft Visual C++ 2015-2022.
