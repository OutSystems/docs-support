OutSystems Developer Cloud 2026-06-22

Released on Jun 22, 2026

New

  • The Traces console now lets you filter traces by agent. Pick an agent from the Assets dropdown and the console returns the traces where that agent is the origin of the request. Apps and agents sit side-by-side in the same control, so you can switch workload types without leaving the page.

    One detail on scope. A trace is filtered by its origin asset. An agent is the origin only when its execution starts from an Event Handler, a Timer, or a REST Expose. When an agent is called from an app, it runs as a span inside that app's trace, so you'll find those runs by filtering for the app.

    (RDASS-3877)

  • Enhanced security by enforcing single-use refresh token rotation globally. Once a refresh token is used to obtain a new access token, it is immediately invalidated and replaced, preventing unauthorized reuse.

    (RDPIB-3180)
