OutSystems Developer Cloud 2025-10-29

Released on Oct 29, 2025

Bug Fixing

  • Fixed a vulnerability where an API Client could still obtain new authentication tokens with an expired secret, allowing unauthorized access to services. The token issuance logic has been updated to correctly verify secret expiration, ensuring clients with expired secrets can no longer receive new tokens. (RDPIB-2904)
  • Fixed an issue on the User Management screen where applying the Active filter could incorrectly display users with a Pending Registration status. The filter now accurately returns only users matching the selected status. (RDPIB-2966)
