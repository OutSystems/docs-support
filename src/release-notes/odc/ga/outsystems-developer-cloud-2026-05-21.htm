OutSystems Developer Cloud 2026-05-21

Released on May 21, 2026

New

  • Workflow Editor: Compare Revisions

    • You can now compare two workflow revisions side by side in the Workflow Editor. The comparison view highlights differences directly on the canvas using color coding: green for added elements, blue for modified or deleted ones, and red for conflicts. Click any highlighted node or connector to inspect property-level changes.
    • You can access the comparison view from three places:
      • Revision history - Compare any previous revision against your current work or the latest published version.
      • Opening a workflow - If a colleague published a new revision while you had unsaved changes, a dialog prompts you to compare both versions before choosing which one to continue editing.
      • Publishing - If a newer revision was published while you were editing, compare your version against the latest published one before deciding to publish or discard your changes.
