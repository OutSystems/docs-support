OutSystems Developer Cloud 2025-06-06

Released on Jun 06, 2025

New

  • [Mentor AppGen]
    • UI Diversity Enhancements: Introduced a new detail screen template, detail with sidebar and tabs, which becomes available when the entity contains a relationship with more than one entity and a high number of attributes, improving the information organization and navigation for complex data models.
    • Enterprise integrations Enhancements: Mentor-generated apps can now write data to external systems such as Salesforce, SQL databases, and SAP. This enhancement streamlines bi-directional data flow and boosts integration capabilities
      • Guidelines:
        • Write operation for external entities will be available only when the entity has the createOrUpdate action available
        • Delete records from external entities will be available only when the entity has the delete action available
