ODC Studio 1.3.0

Released on Oct 24, 2023

New in ODC Studio 1.3.0

  • The new versioning feature for libraries enables an independent lifecycle for each library. If a new version is released, consumers can decide whether and when to adopt the update based on their needs and preferences. You can find more information on the libraries documentation. (RDPMBT-3675)

Bug Fixing

  • Fixed an issue when selecting text inside text editors. (RDOIBT-1866)
  • Fixed an issue which caused crash when user create script from the suggestion to fix a required script error shown in the bottom pane. Now, users can use the suggestion to create script and add it as a required script. (RDPMBT-3652)
