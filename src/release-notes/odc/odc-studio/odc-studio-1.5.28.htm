ODC Studio 1.5.28

Released on Jun 30, 2025

New in ODC Studio 1.5.28

  • Within the Mobile tab, introduced the ability to control the color scheme (Light, Dark, or System) for the status bar content color for Cordova-based applications, for Android only. (RDMDX-74)

Bug Fixing

  • Fixed an issue that had hidden the referenced roles in the scope tree of the expression editor. (RDEV-8511)
  • Fixed an error that could occur when trying to undo the creation of an empty filter in a Dataset. (RDEV-8562)
  • Fixed an issue that caused unexpected error on performing undo multiple times when undo merge dialog is shown. (RDEV-8566)
  • Fixed an issue that caused an unexpected error when loading property dropdown values. (RDEV-8570)
  • "Open in Browser" commands from Screen's context menus now perform local validation only to determine if they're enabled. This change eliminates small delays when you open these menus, enhancing the experience and performance in this action. (RDEV-8571)
  • Fixed an issue when trying to create templates from libraries not displaying any error to the developer. (RDEV-8581)
