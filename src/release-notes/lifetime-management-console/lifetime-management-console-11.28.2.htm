LifeTime Management Console 11.28.2

Released on Feb 25, 2026

New in LifeTime Management Console 11.28.2

  • In "Expose Entities", it is now possible to select a different target environment when promoting entity views within an infrastructure. Target environments must have a higher order than the source environment, and backward promotion is not supported. (R11CT-4632) Application Lifecycle LifeTime
  • In "Expose Entities", it is now possible to filter the list of displayed environments using environment filters created by users. Note that environment filters must be created in the "Applications_List" screen. (R11CT-4633) Application Lifecycle LifeTime

Bug Fixing

  • Fixed an issue where users with no permissions were redirected to the Internal Error instead of the No Permissions screen. The issue occurred when a user without permissions accessed the AuditLogs screen. Now, the user is redirected to the proper NoPermissions screen. (RPM-6441) Application Lifecycle LifeTime
