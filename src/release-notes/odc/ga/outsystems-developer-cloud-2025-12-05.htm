OutSystems Developer Cloud 2025-12-05

Released on Dec 05, 2025

New

  • Previously, inactive OIDC/SAML users could authenticate and receive tokens. Now, these users are blocked at login and receive an unauthorized error. Additionally, logged-in users who are deactivated will lose access on their next refresh. (RDPIDT-3203)
  • Monthly Security Update (November 2025): Updated application container base image, including security fixes. Check Microsoft .NET 8.0.22/8.0.22 release for more details: https://github.com/dotnet/core/blob/main/release-notes/8.0/8.0.22 (RRCT-6691)
