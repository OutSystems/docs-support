OutSystems Developer Cloud 2026-01-26

Released on Jan 26, 2026

New

  • [Mentor AppGen]
    <ul>
    <li> Added support for a new Accordion UI pattern in List Screens. You can now use expandable list items to organize content more efficiently in your mobile and web applications.</li>
    <li>You can now use Expand and Collapse functionality for filters. This improvement helps optimize screen real estate by allowing users to hide filter options when they are not in use.</li>
    <li>Mentor now supports inline editing within these Master-Detail views, allowing you to update records without losing the list context, significantly improving operational efficiency.</li>
    </ul> (RDMOR-3184)

Bug Fixing

  • [Mentor AppGen]
    <ul>
    <li> Fixed an issue where imported entities would not persist in the application model after the preview step. (RDMOR-3076)</li>
    <li> Fixed an OML generation error that occurred during the creation of the Icons Dictionary. This issue was preventing the final application binary from being generated when specific icon sets were used.(RDMOR-3004).</li>
    <li> Resolved an OML generation error that happened when an application specification lacked a List or Dashboard screen. Mentor now correctly generates valid application files even for simplified app specs.(RDMOR-2996)</li>
    <li> Fixed an issue where Mentor failed to suggest external entities even when explicitly requested by the user during the chat. Entity suggestion logic has been fine-tuned for better alignment with user intent.(RDMOR-3045)</li>
    <li> Resolved a crash in app generation triggered by excessively long action names. AppGen now supports standard length constraints for all generated actions.(RDMOR-3033)</li>
    <li>Fixed a bug where Mentor was unable to locate external database connections during the initial app setup. Connection discovery is now more reliable.(RDMOR-3011)</li>
    </ul> (RDMOR-3187)
