LifeTime Management Console 11.28.4

This is a preliminary version of the document.

Bug Fixing

  • Fixed an issue in LifeTime that caused modules not to display their most up-to-date version in the Deploy Custom selection after clicking the Refresh Application button. This occurred when the deployment option was set to Deploy Custom. Reported by Evert.

    (R11CT-3542) Application Lifecycle LifeTime
  • Fixed an issue where modules excluded from a deployment plan (because their version matched in source and target at the time the plan was saved) were not re-evaluated during a plan refresh if a hotfix had been published directly to the target environment. As a result, the deployment would skip those modules and leave an outdated version in the target environment. (RPM-5855) Application Lifecycle LifeTime
