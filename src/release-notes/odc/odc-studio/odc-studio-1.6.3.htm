ODC Studio 1.6.3

Released on Oct 20, 2025

New in ODC Studio 1.6.3

  • It is now possible to create Agents under the Automations tab and re-use them in different logic flows of Server actions. (RDEV-8860)

Bug Fixing

  • Fixed an issue that caused an unexpected error when selecting data types in the searchable Data Type picker dialog. (RDEV-8810)
  • Fixed an issue where updates to descriptions of structure attributes and action arguments in agents were not automatically synced when using inline records. (RDEV-8819)
