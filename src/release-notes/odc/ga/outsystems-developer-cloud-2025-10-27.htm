OutSystems Developer Cloud 2025-10-27

Released on Oct 27, 2025

New

  • In the ODC Portal, it is now possible to undeploy an app from any stage. Undeploying an app removes it from the selected stage but does not delete it from your Organization. (RDLOT-3887)

Bug Fixing

  • Fixed an issue where the deployed revision was not displayed correctly during the deployment of a workflow. (RDLOT-3981)
  • Custom Dropdowns now always open with the selected option correctly at the top of the list. (RPM-6122)
  • Upload widget can now be selected by cycling through it with the tab key and opened with the space or enter key. (RPM-6124)
  • Fixed an issue while using multiple slashes on custom URLs on Screens (RPM-6175)
