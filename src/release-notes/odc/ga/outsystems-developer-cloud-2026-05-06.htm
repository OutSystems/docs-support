OutSystems Developer Cloud 2026-05-06

Released on May 06, 2026

New

  • Workflow Editor: Monaco Editor

    • The expression editor in the Workflow Editor now uses Monaco Editor (the same engine used in Visual Studio Code).
    • This upgrade brings syntax highlighting, intelligent auto-complete, hover tooltips, and a documentation panel, giving developers a significantly improved experience when writing expressions in workflows.

    (RDODCT-4219)

  • Workflow Editor: Horizontal

    • The Workflow Editor now supports horizontal orientation, providing more flexibility in how you visualize and build your business processes.
    • Use the new orientation icons in the canvas toolbar to toggle between the classic vertical layout and the new horizontal mode.
    • Choose the orientation that best fits your screen real estate or the specific complexity of your workflow logic.

    (RDODCT-4236)
