OutSystems Developer Cloud 2026-06-25

Released on Jun 25, 2026

New

  • The Azure SQL Connector now supports Microsoft Entra ID (formerly Azure Active Directory) authentication using Service Principal. This enables identity-based access to Azure SQL databases, aligning with organizational security policies managed through Entra ID. Note that this authentication method is not supported when connecting through a Private Gateway. For setup instructions, see Create connections to external data sources.
