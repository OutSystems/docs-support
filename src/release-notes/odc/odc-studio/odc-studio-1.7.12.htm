ODC Studio 1.7.12

Released on Jun 22, 2026

Bug Fixing

  • Fixed an issue where content copied in the editor could not be pasted back into the application.

    (RDEV-9657)

  • Fixed an issue where Compare and Merge could cause an unexpected error when comparing references.

    (RDEV-9727)

  • Fixed an error that could cause Studio to crash while completing certain editing operations.

    (RDEV-9791)
  • Fixed an issue that incorrectly allowed copy/pasting input parameters from/to different type of parent elements. (RPM-5944)
  • Fixed an issue where some references to libraries were still being shown after removing the reference to the library. (RPM-6271)
  • Fixed an issue where Mentor was generating invalid applications with REST Expose invalid payload. (RPM-6853)
  • Fixed an issue where an Unexpected Error would be shown when attempting to remove an existing database event from an entity within an Agentic application. (RPM-6907)
