Cross-platform Service Studio 11.55.16

Released on Apr 07, 2025

Bug Fixing

  • Fixed an issue that caused an unexpected error when a communication error occurs while downloading modules. (RDEV-8403)
  • Fixed an issue that caused errors starting a debug session, specifically 'ConsumerNoDebugPermission' and 'ProducerNoDebugPermission' errors. (RDEV-8406)
  • Fixed an issue that could introduce a long delay in the Compare and Merge. (RDEV-8413)
  • Fixed an issue where a list from an output parameter of a data action was being allowed to be used in another data action, leading unexpected results in runtime (RPM-5455) Application Runtime Logic Execution
