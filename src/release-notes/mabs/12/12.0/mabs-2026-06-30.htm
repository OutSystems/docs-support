MABS 2026-06-30

Released on Jun 30, 2026

New

  • Fixed an issue in MABS where providing a non-certificate value for the ApplicationSignerCertificate AppShield preference caused builds to fail with a generic internal error (OS-MABS-SHLD-50000). Builds now fail early with a descriptive error (OS-MABS-SHLD-40003) when the preference value is not a valid base64-encoded X.509 certificate.

    (RNMT-8020)
