ODC Studio 1.5.9

Released on Feb 17, 2025

New in ODC Studio 1.5.9

  • Improved the organization connection preferences. (RDEV-8125)
  • When undoing a merge operation, a dialog is now shown warning users of the potential impact. (RDEV-8179)
  • Always show merge override dialog when the publish includes the undo of merges. (RDEV-8181)
  • Removed the 'Override with this revision' button when undoing a merge operation to prevent accidental code loss. (RDEV-8182)

Bug Fixing

  • Fixed an issue that could cause unresponsiveness when refreshing dependencies. (RDEV-8279)
  • Fixed an unexpected error that occurs while closing a tab and performing 1CP. (RDEV-8296)
