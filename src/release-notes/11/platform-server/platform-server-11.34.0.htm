Platform Server 11.34.0

Released on Mar 31, 2025

New in Platform Server 11.34.0

  • Configuration Tool & Service Center will now issue a warning when connecting to an unsupported Oracle version. (R11BRT-1012)
  • Updated Oracle Managed Driver to version 19.26.0. See the Oracle release notes, for more information on related fixes. (R11BRT-1619)
  • Added support for OAuth2 authentication in Platform SMTP. (R11DT-2574)
  • Removed the usage of deprecated unload event in the App Feedback System Component. (R11DT-2776)
  • Removed the usage of deprecated unload event in the PreviewInDevices System Component. (R11DT-2779)
  • Improved the deployment of DDL scripts to mitigate an issue that caused locks in tables that Expose Process Events when publishing. (RPM-5314) Publish Operation Deployment stage

Bug Fixing

  • Fixed incorrect information in the confirmation message displayed when changing the API URL in Service Center. Previously, the message stated that all modules would be reloaded after applying the change. It now correctly indicates that modules will be reloaded only if necessary. (R11CT-3745)
  • Fixed an issue where Structures or Entities consumed from other modules would not load their dependencies when used exclusively as a list Element Type in a given screen. (RPM-1847) Application Runtime Data Access and Manipulation
  • Optimized the compilation and validation of SQL Advanced Queries, to prevent potential timeout errors. (RPM-2216) Publish Operation Compilation
  • Fixed a bug that showed the error message “Cannot read properties of undefined (reading IndexOf)”. This occurred in runtime when redirecting or navigating between screens, in a reactive or mobile app, where the destination screen had a custom URL configured with "Path" as the URL Structure and a variable as an input parameter of any type different than Text or Number. (RPM-3293) Application Runtime Logic Execution
  • Fixed an issue where System logs were visible to Service Center users with limited permissions. (RPM-5225) Application Lifecycle Service Center
  • Fixed an issue with reference local entities when renamed in module, or in aggregate, when fetching data (RPM-5285) Application Runtime Data Access and Manipulation
  • Fixed an issue that caused Triggers to be dropped and recreated when an Entity with Expose Process Events has old columns still in the physical table. (RPM-5373) Publish Operation Compilation
  • Fixed an issue where the DropDownMenu component did not display its arrow. This occurred when the MenuItem placeholder included text wrapped in a span element, caused by adding inline HTML attributes to that element. (RPM-5509) Application Runtime System Components
  • Fixed an issue where Logical Database Configurations would not be set when publishing a solution for the first time. This change may require updates to module references. Check OutSystems 11 side effects and breaking changes for more details. (RPM-5571) Publish Operation Deployment stage
  • Fixed a bug that caused "No role validation found" error messages when calling Client Action from a JavaScript node. (RPM-607) Application Runtime Logic Execution
