OutSystems Developer Cloud 2026-05-11

Released on May 11, 2026

New

  • You can now use the Snowflake connector in ODC Data Fabric to connect your ODC apps and AI agents directly to the Snowflake Data Cloud, without building custom REST API integrations or middleware. (RDCIST-4313)
  • You can now use the Azure AI Search connector with Private Gateway to connect your ODC apps to Azure AI Search services hosted in private networks, without exposing them to the public internet. (RDCIST-4327)
