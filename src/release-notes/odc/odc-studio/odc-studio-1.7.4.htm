ODC Studio 1.7.4

Released on Apr 20, 2026

New in ODC Studio 1.7.4

  • You can now track your changes by adding comments to each revision before publishing. (RDEV-9287)
  • Changed icon colors and slightly increased icon size in Pick an Icon dialog, to enhance readability. (RDEV-9327)

Bug Fixing

  • Improved the feedback sent back to Mentor on application change failures. (RDEV-9414)
  • Fixed an issue where opening element of a library asset from "Add public elements" and the "Open reference" menu failed to focus an existing open tab of the same revision, resulting in redundant tabs. (RDEV-9416)
  • Fixed an issue where Mentor was unable to remove comment nodes with slashes. (RDEV-9431)
  • Fixed an issue where Mentor was unable to apply changes to event handlers and child collections such as OnClick. (RDEV-9439)
  • Fixed an issue where Mentor was unable to resolve REST actions. (RDEV-9441)
  • Fixed an issue where Mentor was not aware of user manual changes on the app between turns. (RDEV-9470)
  • Fixed an issue that caused Mentor to fail applying code changes when handling events in web blocks. (RDEV-9475)
