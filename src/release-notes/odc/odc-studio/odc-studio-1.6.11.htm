ODC Studio 1.6.11

This is a preliminary version of the document.

New in ODC Studio 1.6.11

  • Added the ability to convert Cordova-based extensibility configurations into Universal Extensibility directly within ODC Studio (RDMDX-104)
  • Developers now have access to auto-complete and syntax validation when working with Universal Extensibility configurations on their mobile apps (RDMDX-60)
  • You can now create Mobile Plugins using Mobile Libraries, and existing libraries with mobile extensibility are automatically converted to Mobile Libraries (RDMDX-98)

Bug Fixing

  • Fixed an issue that caused an unexpected error after selecting the "Built-in static entities" or "Built-in structures" folder during merge. (RDEV-9010)
