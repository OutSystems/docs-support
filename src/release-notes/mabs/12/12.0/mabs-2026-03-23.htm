MABS 2026-03-23

Released on Mar 23, 2026

Bug Fixing

  • Fixed an intermittent crash on iOS when HTTP requests were in-flight during a WebView reload, such as after an over-the-air update. (RDMR-1154)
  • Fixed a bug on Android OutSystems capacitor apps that prevented back navigation in scenarios where navigation via deeplink occurred (RDMR-1159)
  • Remove Camera permission from base Android apps (RPM-6607)
  • Fixed a bug on iOS OutSystems capacitor apps preventing images from being loaded when content BinaryData was fetched via server Entity or Server Action (RPM-6658)
