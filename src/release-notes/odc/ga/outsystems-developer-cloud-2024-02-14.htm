OutSystems Developer Cloud 2024-02-14

Released on Feb 14, 2024

New

  • Developers can now select how Null values are handled on their integrations using external connections. Keep Null or Overwrite Null values with default values configurations are now available in Connections, on the ODC Portal. (RCNT-6936)
  • The error code, when applicable, is now displayed and the begging at log message, both in the logs list and log detail page. (RDLOT-2460)
