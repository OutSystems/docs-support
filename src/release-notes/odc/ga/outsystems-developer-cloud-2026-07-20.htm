OutSystems Developer Cloud 2026-07-20

Released on Jul 20, 2026

New

  • Organizations with network-restricted O11 environments can now extend to ODC: create a Private Gateway connection in the ODC Portal to securely route REST API requests from ODC apps to O11 logic, maintaining existing security policies. Requires LifeTime v11.29+. In the ODC Portal, go to Management > OUTSYSTEMS 11 > (your infrastructure) > Secure connectivity > Add. For more information, refer to Logic interoperability. (RDUBR-365)
