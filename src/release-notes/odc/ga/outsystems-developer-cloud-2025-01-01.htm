OutSystems Developer Cloud 2025-01-01

Released on Jan 01, 2025

Bug Fixing

  • The like builtin function in JS now behaves properly with multiline strings and multiline search patterns, the % and _ wildcards will match line breaks like they do in SQL. The LIKE SQL operator also behaves the same way with multiline strings in PWAs and Native Apps now. (RAR-3137)
  • Time outs for Server Actions, Data Actions and Server Aggregates now throw the appropriate error on iOS instead of failing silently. (RAR-3139)
