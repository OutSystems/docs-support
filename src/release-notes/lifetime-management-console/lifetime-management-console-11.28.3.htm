LifeTime Management Console 11.28.3

Released on Apr 27, 2026

New in LifeTime Management Console 11.28.3

  • You can now dismiss the information alert about system entities being exposed by default in the Expose Entities Manage Access screen. Once dismissed, the alert will not appear again for that user. (R11CT-4666) Application Lifecycle LifeTime

Bug Fixing

  • Fixed a Firefox-specific issue where the browser's autofill feature triggered an infinite save loop on the effective value input of secret site properties. This occurred during the 'Configure applications settings' step of a deployment, where the browser repeatedly injected credentials into the 'Secret' input fields. Issue reported by João Heleno. (R11CT-3513) Application Lifecycle LifeTime
  • Fixed an issue that caused LifeTime to display the deployment progress screen indicating the deployment was in progress, when the deployment plan had not yet started. This occurred when navigating directly to the deployment progress screen before the deployment had been initiated. Now, users are redirected to the staging edit step and an informative message is displayed explaining the redirection. Issue reported by Nuno Reis. (R11CT-4277) Application Lifecycle LifeTime
  • In the "Expose Entities" feature, system entity views (User and Tenant) are no longer incorrectly listed. This occurred in infrastructures with Platform Server 11.41.0, where these entities are exposed by default. Now, only user-defined views are displayed. (R11CT-4748) Application Lifecycle LifeTime
  • Removed the low database space warning from the database image used in Enterprise PaaS infrastructures, as the warning text itself was deprecated and removed in LT 11.25.0. (RPM-5490) Application Lifecycle LifeTime
  • Fixed a vulnerability related with service account JWT token. CVSSv3.1 score 2.7 (Low). (RPM-6559) Application Lifecycle LifeTime
