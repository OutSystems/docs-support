OutSystems Developer Cloud 2026-01-23

Released on Jan 23, 2026

New

  • Workflow Editor: Clone Workflow - is available again
        - You can now clone an existing workflow directly from the Workflow Editor.
         - To use this feature, open a workflow in Edit Mode within the Workflow Editor.
         - Click the "..." menu in the top toolbar and select "Clone Workflow" to create a duplicate of the current workflow.
         - This feature was temporary disabled but it has being is enabled again. (RDODCT-3806)
