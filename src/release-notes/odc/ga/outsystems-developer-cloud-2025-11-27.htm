OutSystems Developer Cloud 2025-11-27

Released on Nov 27, 2025

New

  • [Mentor AppGen Enhancements]

    <ul>
    <li> AppGen now generates field-level validation rules for common scenarios, including mandatory fields, past/future date checks, value range enforcement (Numeric, Text, Phone Number), and email validation, based on your app context or explicit requirements.</li>
    <li>Mentor can now include Pop-up and Tab patterns when generating new screens, providing more flexible and modern UI designs.</li>
    <li>We've implemented Popups ViewMode, allowing you to use Edit and View screens within pop-up modals for a faster, more focused user experience.</li>
    <li>Added support for more dashboard aggregations, including Minimum, Maximum, Average, and Sum, providing richer data visualization options out of the box.</li>
    </ul> (RDMAG-876)
