Platform Server 11.40.0

Released on Dec 22, 2025

New in Platform Server 11.40.0

  • Improved the entity-level interoperability between ODC and O11. (R11BRT-2117)

Bug Fixing

  • Fixed an issue where uploaded images did not retain their original quality after being downloaded from the database. (RPM-4008)
  • Fixed a bug that did not allow a list of emails to be passed as an argument to the emailAddressValidate client-side built-in function. (RPM-4832)
  • Fixes Two Stage deployments locking BPT processes during first stage. (RPM-5934)
  • Fixed an issue where Label widgets would not display the mandatory indicator (asterisk) when linked to mandatory Upload widgets. This applies to Reactive and Mobile apps. (RPM-6043)
  • Fixed a security issue where client-side data, including the screen cache, persisted after the session ended or the user logged out. (RPM-6117)
  • Fixed a security vulnerability in an underlying third party component used by the platform. The vendor's recommended updates have been integrated into the platform. No customer action is required beyond applying the standard platform update. (RPM-6287)
