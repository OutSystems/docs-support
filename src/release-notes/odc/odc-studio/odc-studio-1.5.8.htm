ODC Studio 1.5.8

This is a preliminary version of the document.

New in ODC Studio 1.5.8

  • Users are now able to populate a Static Entity with database records using the "Import records from database" button. (RDEV-8169)

Bug Fixing

  • Fixed an issue preventing the import of entities from XLSX files. (RDEV-7975)
  • Fixed an issue that caused an unexpected error when opening an app from the chevron icon while creating a screen by dragging an entity in UI Flow. (RDEV-8275)
  • Fixed an issue that caused an unexpected error when changing the name of an attribute after performing an undo. (RDEV-8280)
